Cuando crea un destino de publicación de YouTube, una de las propiedades predeterminadas que se pueden establecer es la configuración de privacidad de YouTube. De forma predeterminada, hay tres opciones: Público, Privado y No incluido en la lista. Cuando los videos se publican en YouTube, la configuración de privacidad de todos los videos se establecerá en la configuración seleccionada.

Es posible que tenga casos en los que la configuración de privacidad de YouTube para cada video deba ser diferente. En este caso, se puede usar un campo personalizado asociado con el video para controlar la configuración de privacidad, lo que permite que cada video tenga una configuración de privacidad diferente. Cuando el video se publica en YouTube usando Brightcove Social, la configuración de privacidad se establecerá en el valor especificado por el valor del campo personalizado.

Creando el campo personalizado

Nota: Los campos personalizados solo están disponibles para cuentas Professional y Enterprise.

Comience agregando un campo personalizado a su cuenta, yt_privacy_status de tipo Lista. Para obtener información sobre cómo agregar campos personalizados a una cuenta, consulte Creación de campos de metadatos personalizados. La configuración del valor de este campo personalizado controla la configuración de privacidad del video de YouTube:

yt_privacy_status Configuración de privacidad de videos de YouTube
privado privado
público público
no incluido en listado no incluido en listado
el valor es nulo o no establecido público

Consejo: Usando un Lista escriba para el campo personalizado, los usuarios podrán seleccionar el estado usando una lista desplegable en lugar de tener que escribir un valor en.

Asignar un valor al campo personalizado

Una vez que se ha agregado el campo personalizado, el valor se puede establecer usando el módulo de Medios.

Editando el destino de publicación

Edite el destino de publicación de YouTube y configure el Configuración de privacidad predeterminado a Utilice el campo personalizado YT_Privacy_Status.

Publicando el video

Cuando se selecciona un video y se selecciona el destino de YouTube, tendrá el estado de privacidad que se estableció mediante el campo personalizado.
