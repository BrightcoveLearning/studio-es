Después de revisar todos los temas de este Inicio rápido, debería poder:

  • Navegar dentro de Video Cloud Studio
  • Subir videos a Video Cloud
  • Editar propiedades de video
  • Crea y diseña un solo reproductor de video
  • Publica un video en la web
  • Crea listas de reproducción
  • Crear un reproductor de listas de reproducción
  • Publica una lista de reproducción en la web.

Audiencia

Cualquiera que esté evaluando o sea nuevo en Video Cloud.

Prerrequisitos

  • Una cuenta de Video Cloud: puede registrarse para obtener una cuenta de prueba aquí.
  • Debería tener un video de muestra que pueda usar o puede descargar algunos videos de muestra

Esquema

Cada tema de este esquema cubre una parte específica del flujo de trabajo de publicación de videos usando Video Cloud. Cada tema tiene un video de descripción general seguido de pasos prácticos que puede seguir. Si prefiere ver todos los videos de capacitación y no completar ninguno de los ejercicios prácticos, consulte el Conceptos básicos de Video Cloud curso de formación bajo demanda.

  • Subir un video
  • Editar propiedades de video básicas
  • Crear un reproductor
  • Diseñar un reproductor
  • Publicar un solo video en la web
  • Creación de una lista de reproducción manual
  • Crear una lista de reproducción inteligente
  • Creación de un reproductor de listas de reproducción
  • Publicar un reproductor de listas de reproducción

El módulo Video Cloud Analytics no se trata como parte de este Quick Start. Para obtener más información sobre los análisis de Video Cloud, vea el Introducción a Video Cloud Analytics curso de formación bajo demanda.

¿Qué debería hacer después?

  • Para profundizar en Video Cloud, vea el Introducción a Video Cloud curso de formación bajo demanda.
  • Para conocer los tipos de análisis de video que están disponibles, vea el Introducción a Video Cloud Analytics curso de formación bajo demanda.
  • Para aprender a publicar portales de video o experiencias en la página usando la Galería, vea la Construyendo experiencias de portal usando la Galería o Creación de experiencias in-page mediante la Galería formación bajo demanda.
