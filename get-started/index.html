Empezando

Video Cloud Studio le permite administrar su flujo de trabajo de publicación de videos en computadoras de escritorio, tabletas y dispositivos móviles. Ofrece una interfaz fácil de usar para cargar, administrar y publicar sus videos. Esta página describe los conceptos clave para ayudarlo a comprender y tener éxito en el uso de Video Cloud Studio para administrar su flujo de trabajo de video. Para obtener más información, consulte los siguientes conceptos:
