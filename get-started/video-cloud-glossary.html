A

Medios activos
La cantidad de videos activos en la cuenta. Tenga en cuenta que estos datos no son en tiempo real, se sincronizan desde su biblioteca de videos una vez al día. También tenga en cuenta que estos datos pueden no ser precisos si está mirando datos analíticos "en tiempo real" (menos de 3 días).
API de análisis
Interfaz de programación de aplicaciones basada en REST que permite a los desarrolladores recuperar métricas analíticas para videos de Video Cloud directamente en formato JSON. los API de análisis proporciona los datos para la interfaz de usuario de Analytics en Video Cloud Studio.
Token de acceso
Un token OAuth2 temporal que permite el acceso a las API de REST, como la API de CMS, la API de análisis y la API de ingesta dinámica.
Afiliado
Editores web que distribuyen reproductores de Video Cloud.
Solicitud de anuncio
Una solicitud (llamada) realizada desde un reproductor de Video Cloud a un servidor de anuncios. La solicitud de anuncio es una llamada HTTP e incluye pares clave / valor que transmiten información que ayuda al servidor de anuncios a orientar un anuncio adecuado para el contexto. No todas las solicitudes de anuncios devolverán un anuncio.
API de análisis
los API de análisis le permite obtener datos analíticos de forma programática para cuentas y videos de Video Cloud.
Apple HLS
Apple HTTP Live Streaming, un protocolo implementado por Apple para entregar video bajo demanda. Apple HLS divide el video en una secuencia de descargas de archivos pequeños, cada descarga carga una pequeña parte del video usando HTTP. Tenga en cuenta que Apple HTTP Live Streaming no tiene nada que ver con transmisión de video en vivo.
Relación de aspecto
La relación entre el ancho y el alto del fotograma de un video. 4: 3 para sistemas estándar y 16: 9 para HDTV.

B

Banda ancha
El rango de frecuencias de señal que un equipo de audio o video puede codificar o decodificar. El video usa una frecuencia más alta que el audio, por lo que requiere un ancho de banda más amplio.
Blog
Una publicación basada en la web que consta de contribuciones periódicas, a menudo en orden cronológico inverso.
reproductor de Brightcove
Brightcove Player es un reproductor de video HTML5 multiplataforma de alto rendimiento que se carga rápidamente, ofrece video HLS de tasa de bits adaptativa de alta calidad en plataformas móviles y de escritorio, y se integra con sus sistemas de análisis y publicidad favoritos.
API de Brightcove Player
La API de Brightcove Player se puede utilizar para controlar mediante programación la apariencia y el comportamiento de Brightcove Player.
Canal de video de banda ancha
Una colección de videos relacionados distribuidos a través de Internet.
Parachoque
Un videoclip corto que puede aparecer antes de un video completo o cuando se carga un reproductor.

C

API de CMS
los API de CMS proporciona acceso de lectura y escritura sin caché a los metadatos del video.
Códec
El algoritmo utilizado para capturar video o audio analógico en forma digital. Comparar con envase.
Envase
Un contenedor de video es un formato que define cómo se almacenan los metadatos del video junto con los datos de video y audio. No define la codificación o compresión de los datos. Por ejemplo, MP4 puede ser un contenedor para archivos H.264.
Red de entrega de contenido (CDN)
Un servicio que ofrece servidores en red que entregan contenido de forma transparente (especialmente contenido multimedia de gran tamaño) a los usuarios finales. Dos CDN prominentes son Akamai y Limelight.
Orientación de contenido
El objetivo de contenido es un proceso que se utiliza para mostrar contenido diferente a diferentes visitantes en función de la información conocida sobre ese visitante.
CSS
Las hojas de estilo en cascada son un mecanismo estándar para controlar el diseño de una página HTML. CSS se utiliza para diseñar Brightcove Players.
Puntos de referencia
Marcadores invisibles insertados en puntos específicos de un video que se pueden usar para activar eventos externos, como una solicitud de anuncio mid-roll o un subtítulo cerrado. Puede establecer puntos de referencia utilizando el módulo de medios.

D

DASH (MPEG-DASH)
Dynamic Adaptive Streaming sobre HTTP (también conocido como MPEG-DASH) es una técnica de transmisión de velocidad de bits adaptativa que permite la transmisión de alta calidad de contenido multimedia a través de Internet entregado desde servidores web HTTP convencionales. Similar a Transmisión en directo HTTP de Apple (HLS) solución, MPEG-DASH funciona dividiendo el contenido en una secuencia de pequeños segmentos de archivo basados en HTTP.
Restricción de dominio
Una forma básica de seguridad de contenido que evita que un reproductor funcione excepto en dominios específicos.
DRM
La gestión de derechos digitales es una forma de cifrar archivos multimedia y empaquetarlos para que solo se puedan descifrar en función de una clave de política.
Entrega dinámica
Ver Transmisión de velocidad de bits múltiple.
API de ingesta dinámica
La API Dynamic Ingest (DI) se utiliza para indicarle a Video Cloud que recupere un activo de video y cree representaciones para él. los API de ingesta dinámica También se puede utilizar para ingerir imágenes en miniatura y pósters o archivos WebVTT para títulos, capítulos o subtítulos.

E

Codificación
Conversión de datos de video y audio en una transmisión que se puede reproducir en un reproductor.
Cifrado
El proceso de codificación de video de tal manera que solo las partes autorizadas puedan verlo.

F

Cuadros por segundo
El número de fotogramas por segundo en el que se muestra un videoclip.

GRAMO

Filtrado geográfico
Imponer restricciones geográficas a los videos, a menudo por motivos de propiedad de los derechos.

H

Formato H.264
También conocido como MPEG-4 AVC (codificación de video avanzada). Un formato de archivo que se usa para enviar video a través de Internet. Se recomienda el formato H.264 para la entrega de video en línea. Los archivos H.264 suelen utilizar el contenedor MP4 y tienen una extensión de archivo.mp4. Ofrece una compresión significativamente mayor que los formatos anteriores.
HLS
Transmisión en vivo HTTP: un formato de video creado por Apple que hace que un video esté disponible en segmentos cortos que se transmiten en el lado del cliente.
HTML
(Lenguaje de marcado de hipertexto) Un lenguaje de marcado diseñado para la creación de páginas web con hipertexto y otra información que se mostrará en un navegador web.
HTML5
HTML5 es un lenguaje de marcado de tecnología central de Internet que se utiliza para estructurar y presentar contenido para la World Wide Web. HTML5 es la quinta revisión del estándar HTML del World Wide Web Consortium (W3C).

I

Perfil de ingesta
Los perfiles de ingesta contienen la configuración de transcodificación que se utilizará cuando se carguen videos mediante el módulo de carga o la API de ingesta dinámica.
API de perfiles de ingesta
La API de perfiles de ingesta le permite crear, recuperar, actualizar y eliminar perfiles de ingesta para su cuenta de Video Cloud. Los perfiles de ingesta definen las representaciones que se crean cuando agrega un video a Video Cloud.

J

JavaScript
Un lenguaje de secuencias de comandos mejor conocido por su uso en sitios web, aunque también se utiliza para permitir el acceso de secuencias de comandos a objetos incrustados en otras aplicaciones.
JPEG (Grupo conjunto de expertos en fotografía)
Un método estándar de compresión con pérdida de uso común que se utiliza para imágenes digitales. JPEG es el formato de imagen estándar que se utiliza en el servicio Brightcove para miniaturas, fotografías de vídeo e imágenes de fondo.

K

Pares clave / valor
Cuando un reproductor envía solicitudes de anuncios, incluye uno o más pares clave / valor (como categoría = deportes) que brindan al servidor de anuncios información sobre el contexto del anuncio y mejoran la orientación del anuncio.

L

Buzón
Un método para mostrar video de pantalla ancha en una pantalla con una relación de aspecto diferente dejando un espacio, generalmente barras negras, encima y debajo de la imagen.
Logo
Un gráfico patentado que representa simbólicamente la marca de una empresa.
Superposición de logotipo
Una pequeña imagen transparente que se muestra en la ventana de reproducción de video. A veces también se denomina gráfico digital en pantalla (PERRO), error o marca de agua.
Bucle / bucle
Cuando se completan los videos de una lista de reproducción y la lista de reproducción comienza a reproducirse desde el principio.
Transmisión de video en vivo
Un protocolo para la transmisión de video de un evento a medida que ocurre, al impulsar transmisiones capturadas y codificadas directamente en lugar de distribuir archivos codificados para transmisión como para VOD.

METRO

Mediateca
La colección de videos, listas de reproducción y otros activos en su cuenta de Video Cloud.
Metadatos
Los metadatos se refieren a las descripciones, etiquetas, géneros y otra información relacionada que proporcionas en la definición de un video.
Sistema de clasificación de películas MPAA
Un sistema, instituido por la Motion Picture Association of America, para calificar una película según su contenido. El sistema de clasificación MPAA (G, PG, R, etc.) es el sistema más reconocido para clasificar contenido potencialmente ofensivo, pero generalmente se restringe su uso en la industria cinematográfica porque la MPAA tiene marcas registradas en cada clasificación individual.
Transmisión de velocidad de bits múltiple
Una función que admita varias versiones, o interpretaciones, para un solo vídeo y permite entregar una copia que mejor se adapte al tamaño y ancho de banda del navegador del espectador. Anteriormente nos referíamos a esto como "entrega dinámica".
Muxing y demuxing
Escritura y lectura de transmisiones de audio y video hacia y desde varios envase formatos.

N

SDK de reproductor nativo
Los SDK de Brightcove Native Player proporcionan las herramientas necesarias para crear aplicaciones habilitadas para video para iOS y Android.

O

OAuth
OAuth es un estándar abierto para la autorización. OAuth proporciona a las aplicaciones cliente un acceso delegado seguro' a los recursos del servidor en nombre del propietario de un recurso.
API de OAuth
La API de OAuth es una API RESTful que le permite obtener tokens de autorización para acceder a otras API REST de Brightcove.

PAG

Reproducción
La reproducción de medios de video en un reproductor de Video Cloud.
API de reproducción
La API de reproducción se puede utilizar en tiempo de ejecución para acceder a datos de listas de reproducción y videos para aplicaciones móviles o portales de video.
API de gestión de reproductores

La API de administración de reproductores se puede utilizar para:

  • Crea un reproductor
  • Ver la configuración de un reproductor
  • Cambiar la configuración de un reproductor
  • Enumere todos los reproductores asociados con un editor
  • Publica un reproductor para ponerlo a disposición del consumidor.
Código de publicación del reproductor
El "fragmento" de código utilizado para incrustar Brightcove Player en una página web.
Lista de reproducción
Una colección de videos que se agrupan en un orden particular para su reproducción en un reproductor de Video Cloud.
API de lista de reproducción
La API de lista de reproducción se utiliza para controlar mediante programación una lista de reproducción mediante JavaScript.
API de política
La API de política se utiliza para crear u obtener claves de política que se pueden utilizar para acceder a la API de reproducción. Las claves de política se generan automáticamente para los reproductores de Brightcove.
Imagen de póster
Una imagen estática que se muestra en el área de reproducción de un reproductor de video antes o después de la reproducción de video.
px
Abreviatura de píxeles utilizados en CSS.

R

Interpretación
Cuando utiliza la transmisión de múltiples tasas de bits, puede crear varias versiones del mismo video, cada una con una tasa de bits y dimensiones diferentes. Cada una de estas versiones se denomina interpretación. La transmisión de múltiples velocidades de bits de Video Cloud elige la mejor interpretación, según las dimensiones del reproductor y la velocidad de conexión del espectador.
Identificación de referencia
Una identificación única asignada por un editor a un video o una lista de reproducción. El ID de referencia (ref id) es distinto del ID del video o del ID de la lista de reproducción, que son ID únicos generados por Video Cloud. El ID de referencia permite la coordinación entre Video Cloud y otros servicios, como el sistema de gestión de contenido de un editor.
RSS Feed
Really Simple Syndication es una tecnología popular para notificar a los usuarios sobre actualizaciones de contenido en un sitio web, blog o canal de televisión por Internet.

S

Buscador
Un programa, como Google, diseñado para ayudar a encontrar información en la web.
Sitio web de la red social
Un sitio web, como Facebook o Twitter, que ofrece a sus miembros espectadores la posibilidad de compartir mensajes, medios y otra información entre ellos.
Transmisión
Un formato de codificación de baja tasa de bits diseñado para su uso en redes e Internet. La transmisión transmite el video en pequeños segmentos con una tasa de bits codificada para que coincida con la velocidad de conexión del usuario, por lo que el espectador remoto puede reproducir audio o video con una interrupción mínima sin descargar primero el archivo de video completo.

T

Etiquetas
Palabras clave o frases individuales para organizar el contenido. Brightcove admite el uso de etiquetas como una forma para que los editores organicen y clasifiquen videos y para crear listas de reproducción.
Miniatura
Una pequeña imagen asociada con un video que se utiliza en los listados del video en reproductores, sitios web y motores de búsqueda de videos.
Autenticación basada en token
Los sistemas de autenticación basados en tokens permiten a los usuarios ingresar su nombre de usuario y contraseña para obtener un token que les permite buscar un recurso específico sin necesidad de su nombre de usuario y contraseña. Una vez que se ha obtenido su token, el usuario puede ofrecer el token, que ofrece acceso a un recurso específico durante un período de tiempo determinado, al recurso.
Pautas para padres de TV
Un sistema de clasificación de contenido establecido por la Comisión Federal de Comunicaciones (FCC) de EE. UU. Para clasificar los programas de televisión según la edad del contenido.

U

URL
Un localizador uniforme de recursos (URL) es una forma de identificar un recurso y su ubicación en Internet (por ejemplo, http://www.brightcove.com).

V

Tasa de bits variable
Un esquema de compresión en el que cada unidad de material de entrada se puede comprimir a diferentes tamaños. Para el video MPEG-2, por ejemplo, esto significa que las secuencias "más fáciles" (es decir, sin movimiento) se pueden comprimir a tamaños muy pequeños, mientras que las secuencias "duras" (con mucho movimiento y cortes de escena) se pueden comprimir a tamaños mucho más grandes. Tamaños.
Video

Un solo título de video para reproducir en un reproductor de Video Cloud. Un video se basa en un archivo de video, que puede comenzar como uno de los muchos formatos de archivo de video admitidos, pero que se transcodifica en un archivo H.264 o archivo FLV si aún no está en uno de esos formatos. Además, un video puede incluir metadatos como:

  • nombre
  • breve descripción
  • descripción larga
  • etiquetas
  • imagen fija de video
  • imagen del cartel
  • parachoque
  • superposición de logotipo
Estudio de Video Cloud

Una aplicación de Internet para acceder y controlar el servicio Video Cloud. El estudio incluye una serie de módulos para realizar diferentes tareas:

  • Subir - Cargue videos a su cuenta de Video Cloud
  • Medios de comunicación - Establecer propiedades de video, organizar videos usando carpetas y publicar videos
  • reproductores - Crear y diseñar reproductores y generar código de publicación para reproductores.
  • Analítica - Ver análisis sobre el rendimiento de los videos y los reproductores
  • Vivir - Crea y transmite eventos en vivo
  • Galería - Crea portales de video para tu contenido de video
  • Audiencia - Conecta análisis de video de Video Cloud a plataformas de automatización de marketing
  • Social - Distribuir videos a YouTube, Facebook y Twitter.
Herramientas de edición de video
Una aplicación de software para editar archivos de video en una computadora personal.
Imagen de póster de video
Una imagen estática que se muestra en el área de reproducción de un reproductor de video antes o después de la reproducción de video.
VOD
Vídeo a la carta. Un protocolo que permite la reproducción de archivos de video distribuidos mediante transmisión o descarga progresiva a voluntad, es decir, bajo demanda. Opuesto a transmisión de video en vivo.

W

WebVTT
Formato de pistas de texto de vídeo web. Los archivos WebVTT proporcionan títulos o subtítulos para el contenido de video, y también descripciones de video de texto, capítulos para la navegación de contenido y, en general, cualquier forma de metadatos que esté alineada en el tiempo con el contenido de audio o video. Los reproductores de Brightcove utilizan subtítulos en formato WebVTT.

X

XLSX
Archivo de Microsoft Excel. Los datos de análisis se pueden exportar en este formato.
