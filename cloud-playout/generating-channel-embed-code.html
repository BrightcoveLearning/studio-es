Generando el código de inserción para un canal de Cloud Playout

Para generar el código de inserción para un canal de Cloud Playout, utilizará el módulo Live. Para obtener detalles completos sobre el uso del módulo Live, consulte Creación y administración de eventos en vivo usando el módulo en vivo.

Hay dos tipos de código de inserción disponibles:

  • Estándar - código de incrustación de iframe. Recomendado para la mayoría de los casos. Este tipo coloca al reproductor dentro de una etiqueta <iframe>.
  • Avanzado - Código de inserción en la página. Permite que el reproductor exista en la página HTML directamente, no en un iframe. Esto ofrece a los desarrolladores los beneficios de acceder fácilmente al reproductor y a las propiedades y eventos asociados.

También tiene la posibilidad de obtener la URL del reproductor publicado. Esto es útil para ver el evento en un navegador.

Para generar el código de inserción para un evento en vivo, siga estos pasos:

  1. Abra el módulo Live.
  2. Busque el evento en vivo al que se transmitirá el canal. Normalmente, el evento se ubicará en el Apoyar pestaña y tener un nombre de Canal a menos que haya seleccionado un evento específico para transmitirlo cuando se creó el canal.
  3. Haga clic en el nombre del evento para abrir el Sala de control página.
  4. Hacer clic Publicar e insertar> Configuración del reproductor en la navegación de la izquierda.
    Nota: Si el canal está configurado para habilitar anuncios del lado del servidor (SSAI), Brightcove Player versión 6.18.3 y superior deber ser usado. Solo los reproductores de la versión 6.18.3 y superiores aparecerán en la lista desplegable.
  5. Haga clic en el URL del reproductor para obtener una vista previa del evento en un navegador o copiar el código de inserción apropiado para usar.

Si el programa fue creado con Inserción de anuncios opción establecida en Sobre , las opciones de publicación incluirán una Seleccionar configuración de anuncios menú desplegable donde se puede seleccionar la configuración de anuncios que se utilizará.
