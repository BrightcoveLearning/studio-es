Exportación de datos de informes

Para exportar datos de un informe, siga estos pasos.

  1. Navegue hasta el módulo de análisis.
  2. Utilice el selector de fecha para seleccionar un intervalo de tiempo.
  3. Seleccione un informe.
  4. Haga clic en el enlace correspondiente para exportar los datos como CSV o XLSX expediente.
  5. Guarde el archivo de datos.
