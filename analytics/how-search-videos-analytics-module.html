Uso de la búsqueda rápida

La función de búsqueda rápida en el módulo de análisis se puede utilizar para buscar un video específico. Después de ingresar los criterios de búsqueda, Búsqueda rápida buscará el nombre del video, la descripción breve, la descripción larga, las etiquetas, la identificación del video y cualquier campo de texto personalizado que pueda tener. Se mostrará una lista de coincidencias debajo del campo de búsqueda.

Tenga en cuenta que eliminado los videos no se pueden buscar a través de la función de búsqueda, pero mantenemos análisis para ellos. Puede encontrarlos expandiéndolos o navegando por el informe de rendimiento y utilizando la función de búsqueda del navegador o descargando el informe y buscando en una aplicación de hojas de cálculo.

Al hacer clic en el enlace a una coincidencia, se abrirá el informe que está "en contexto". Por ejemplo, si está viendo la participación de su cuenta y luego usa la Búsqueda rápida para ubicar un video específico, al hacer clic en el enlace al video, se abrirá el Informe de participación. Si está mirando los informes de rendimiento, al buscar y hacer clic en un enlace se abrirá el informe de rendimiento.

Búsqueda básica

Si proporciona un valor de búsqueda sin calificadores, esto ejecutará una búsqueda de texto en los siguientes campos:

  • Nombre
  • Breve descripción
  • Descripción larga
  • Etiquetas
  • ID de video
  • Identificación de referencia
  • Todos los campos personalizados

Si desea hacer coincidir varias palabras, simplemente envuelva el término entre comillas.

Búsqueda Avanzada

El módulo Analytics también proporciona la posibilidad de buscar campos específicos, buscar en función de la fecha e incluir o excluir términos específicos.

Buscando campos específicos

Para buscar un valor específico en un campo específico, use esta sintaxis:

  • field:value to search

Se admiten los siguientes campos:

Campo Valores admitidos
name cadenas o cadenas entre comillas
text cadenas o cadenas cotizadas (valores en name, short description, long description)
tags cadenas o cadenas entre comillas
reference_id cadena o cadenas entre comillas
state ACTIVE, INACTIVE, DELETED, PENDING
updated_at ver Buscando fechas específicas para formato de fecha
created_at ver Buscando fechas específicas para formato de fecha
published_at ver Buscando fechas específicas para formato de fecha
complete true o false

Si desea hacer una coincidencia exacta o hacer coincidir varias palabras, simplemente envuelva el término entre comillas:

  • "live"
  • "live streaming"

Esto también funciona cuando se busca en un campo específico:

  • name:"live"
  • name:"live streaming"

Buscando fechas específicas

Los campos de búsqueda dependientes de la fecha toman valores de rango de fechas. Se admiten varios formatos para especificar la fecha:

Formato Significado
2013-08-01 Fecha en formato aaaa-mm-dd
2013-08-01T00: 00: 00Z Una hora en formato UTC
-1d Hora actual menos 1 día (ver Fechas relativas)
+1d Hora actual más 1 día (ver Fechas relativas)
 

Fechas relativas

Para fechas relativas, apoyamos una dirección (+ o -) seguida de un número, seguido de una duración (o su abreviatura). Las duraciones legales son: minutos (m), horas (h), días (d)

Por ejemplo:

  • -1day
  • -2days
  • +4hours
  • -60minutes
  • -1d
  • -2d
  • +4h
  • -60m

Rangos de fechas

Si el campo en el que está buscando es una fecha, puede especificar un rango de fechas para buscar:

  • updated_at:2013-08-01T00:00:00Z..2013-10-08T00:00:00Z

Esto buscará todos los videos con un updated_at valor entre el 1 de agosto de 2013 y el 8 de octubre de 2013.

Rangos abiertos

Si desea hacer coincidir todas las fechas hasta una hora determinada, o hacer coincidir todas las fechas desde una hora determinada, deje un extremo del rango. Por ejemplo, para buscar todos los videos modificados en los últimos 2 días:

  • q=updated_at:-2days..

Busque todos los videos modificados antes del 11 de agosto de 2013:

  • q=updated_at:..2013-08-11T00:00:00Z

 

Incluyendo y excluyendo términos específicos

Puede marcar un término como requerido (videos devueltos DEBER coincidencia) o prohibido (los videos devueltos deben NO partido). Esto se controla con un + o - firmar inmediatamente antes del término:

Ejemplo Significado
+animals vídeos DEBE incluir el término «animales»' en name short description, o long description
-animals los vídeos NO deben incluir el término «animales»' en el name, short description , o long description
+name:animals vídeos DEBEN incluir el término «animales»' en el nombre
-name:animals los vídeos NO deben incluir el término «animales»' en el nombre
