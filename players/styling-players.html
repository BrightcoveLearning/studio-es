Algunas opciones básicas de estilo de reproductor se pueden configurar usando el módulo de reproductores. Para una personalización completa de Brightcove Player, debe utilizar Hojas de estilo en cascada (CSS). Para obtener más información sobre el uso de CSS para personalizar la apariencia del reproductor, consulte la Paso a paso: Personalización del reproductor documento de desarrollador.

Para configurar el reproductor Estilismo configuración, siga estos pasos:

  1. Hacer clic Reproductores en el encabezado de navegación para abrir el módulo Reproductores.
  2. Haga clic en el enlace del reproductor que desea configurar.
  3. Hacer clic Estilismo en el menú de navegación de la izquierda.

Se pueden configurar los siguientes ajustes de estilo:

  • Tipo de lista de reproducción
  • Título y descripción
  • Forma del botón de reproducción
  • Posición del botón de reproducción
  • Tema de color
  • Superposición de logotipo

Una vez realizados los cambios, asegúrese de Publicar el reproductor para que los cambios sean visibles.

Nota: Cualquier cambio realizado en un reproductor puede tardar hasta 5 minutos en aparecer en los reproductores publicados.

Tipo de lista de reproducción

La Tipo de lista de reproducción El ajuste controla cómo se muestra la lista de reproducción. Esta opción solo aparecerá si Tipo de reproductor se establece en Lista de reproducción como parte de la Información del reproductor propiedades. Las siguientes opciones están disponibles:

  • Lista de reproducción horizontal - El reproductor mostrará una lista de reproducción horizontal
  • Lista de reproducción vertical - El reproductor mostrará una lista de reproducción vertical
  • Lista de reproducción oculta - La lista de reproducción se ocultará.

Título y descripción

Seleccione Mostrar título y descripción para mostrar el título y la descripción del video en el reproductor.

Forma de botón de reproducción

La Forma del botón de reproducción El ajuste controla la forma del botón de reproducción. Se admiten las siguientes formas:

  • Circulo
  • Cuadrado
  • Rectángulo

Posición del botón de reproducción

La Posición del botón de reproducción El ajuste controla la ubicación del botón de reproducción dentro del reproductor. Se admiten las siguientes posiciones:

  • Centrar
  • Arriba a la izquierda
  • Parte superior derecha

Tema de color

La Tema de color La configuración controla los colores utilizados en el reproductor, incluidos los botones de reproducción / pausa, el tiempo transcurrido y el volumen, los subtítulos, la pantalla completa y los iconos para compartir.

Se pueden configurar los siguientes colores de tema:

  • Principal - Controla el color del botón de reproducción grande, la barra de progreso y el nivel de volumen
  • Fondo - Controla el color de fondo de los controles del reproductor.
  • Iconos - Controla el color de la flecha en el botón de reproducción y los iconos en la barra de control del reproductor

Para cambiar el color de un tema:

  1. Haga clic en un color de tema para abrir el selector de color.
  2. Haga clic en un color en el lado de conducción del control.
  3. Haga clic en el cuadrado para seleccionar un tono del color deseado. El color también se puede configurar utilizando el modelo de color de tono, saturación y valor (HSV) en el control de texto.
  4. El control deslizante debajo del cuadrado de color se puede usar para establecer la opacidad de los controles.
  5. Haga clic fuera del selector de color para modificar otro color de tema.

Superposición de logotipo

Una superposición de logotipo es una imagen que aparece en la parte superior del reproductor. A continuación, aparece una superposición de logotipo de muestra.

Nota: El complemento de superposición también se puede configurar manualmente y proporciona la capacidad de mostrar texto y varias superposiciones. Para obtener información, consulte Configuración de una superposición de reproductor mediante el complemento Superposición.

Para agregar una superposición de logotipo:

  1. Selecciona el Superposición de logotipo a En.
  2. Haga clic en Examinar y seleccione una imagen superpuesta (se admiten imágenes.png, .jpg y.gif).
  3. (Opcional) Ingrese un URL del logotipo. Cuando se hace clic en la superposición, esta URL se abrirá en una nueva pestaña del navegador.
  4. Seleccione un Posición del logotipo para la superposición:
    • Abajo a la derecha
    • Abajo a la izquierda
    • Parte superior derecha
