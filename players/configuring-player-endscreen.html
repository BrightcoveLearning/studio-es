Los reproductores se pueden configurar para que se muestre una pantalla final cuando finalice un video. La pantalla final se puede configurar para mostrar los iconos de reproducción y uso compartido o se puede personalizar para incluir HTML. A continuación se muestra una pantalla final personalizada.

Se muestran iconos para permitir que el espectador vuelva a reproducir el video o lo comparta en sitios populares de redes sociales.

Nota: si se configura la pantalla final de un reproductor, no se mostrará en un dispositivo iOS si el reproductor está en modo nativo de pantalla completa.

Para configurar la pantalla final de un jugador, siga estos pasos:

  1. Hacer clic Jugadores en el encabezado de navegación para abrir el módulo Jugadores.
  2. Haga clic en el enlace del reproductor que desea configurar.
  3. Hacer clic Pantallas finales en el menú de navegación de la izquierda.
  4. Selecciona el Pantalla final del jugador opción. Las siguientes opciones están disponibles:
    • Ninguno: no se mostrará ninguna pantalla final
    • Social - Mostrar el cuadro de diálogo para compartir en redes sociales cuando finalice el video
    • Personalizado - Muestra una pantalla final personalizada cuando finaliza el video
Nota: Cualquier cambio realizado en un jugador puede tardar hasta 5 minutos en aparecer en los reproductores publicados.

Pantalla final social

Cuando el Social está seleccionada la opción, el cuadro de diálogo para compartir en redes sociales se mostrará cuando finalice el video.

El uso compartido en redes sociales del jugador debe estar habilitado para usar la pantalla final de uso compartido en redes sociales. Si el Social está seleccionada la opción, el intercambio social se habilitará automáticamente. Las opciones para compartir en redes sociales que se muestran se basan en lo que se configuró como parte de la configuración social. Para obtener más información sobre la configuración social, consulte Configurando lo Social Propiedades para un jugador.

Pantalla final personalizada

Cuando el Personalizado Si selecciona esta opción, se puede utilizar HTML personalizado para mostrar texto o imágenes adicionales encima de los iconos de reproducción y uso compartido. Para configurar una pantalla final personalizada, seleccione Personalizado y pegar en HTML para que se muestre en el HTML personalizado propiedad.

Nota: Se puede utilizar un vídeo añadiendo el elemento <vídeo> html a HTML personalizado.
Nota: El uso compartido debe estar habilitado para que el icono de uso compartido aparezca en la pantalla final.
Nota: Si el Personalizado está seleccionada la opción y no se proporciona HTML, el botón de reproducción se mostrará cuando finalice el video.
