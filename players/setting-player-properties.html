Para configurar o editar la configuración del jugador, abra el módulo de Jugadores y haga clic en el nombre de un jugador. Para editar las propiedades del reproductor, haga clic en un elemento del menú de navegación de la izquierda.

Se pueden configurar las siguientes propiedades del reproductor:

  • Información del jugador
  • Estilismo
  • Reproducción
  • Control S
  • Social
  • Complementos
  • Pantallas finales
  • Publicidad
  • Restricción de contenido
  • Campaña
  • Editor JSON
  • Análisis de terceros
Nota: Cualquier cambio realizado en un jugador puede tardar hasta 5 minutos en aparecer en los reproductores publicados.

Información del jugador

La Información del jugador Las propiedades se utilizan para configurar el nombre del reproductor, el tamaño, la relación de aspecto, el modo de actualización de las dimensiones y la versión. Para más información, ver Configuración de las propiedades de información del jugador.

Estilismo

La Estilismo Las propiedades se utilizan para configurar el tipo de jugador, la apariencia del jugador, los colores y la superposición de un jugador. Para más información, ver Jugadores de estilo.

Reproducción

La Reproducción Las propiedades se utilizan para configurar las opciones de reproducción de video. Para más información, ver Configuración de las propiedades de reproducción del reproductor.

Control S

La Control S Las propiedades se utilizan para configurar los controles que aparecen en la barra de control del reproductor. Para más información, ver Configurar los controles del jugador.

Social

La Social Las propiedades se utilizan para configurar las opciones sociales del jugador. Para más información, ver Configuración de las propiedades sociales de un jugador.

Complementos

Complementos utilice una combinación de HTML, JavaScript y / o CSS para personalizar el comportamiento del reproductor. Los complementos se pueden desarrollar para:

  • Modificar el comportamiento predeterminado de un jugador
  • Agregar funcionalidad a un reproductor
  • Personalizar la apariencia del jugador

Para más información, ver Configuración de complementos del reproductor.

Pantallas finales

La Pantallas finales Las propiedades se utilizan para configurar lo que se muestra cuando finaliza un video. Para más información, ver Configuración de la pantalla final del reproductor.

Publicidad

La Publicidad Las propiedades se utilizan para habilitar anuncios IMA en un reproductor. Para más información, ver Configurar la publicidad del jugador.

Restricción de contenido

La Restricción de contenido Las propiedades se utilizan para configurar restricciones de dominio y filtrado geográfico para el reproductor. Para más información, ver Configuración de restricciones de contenido del reproductor.

Campaña

los Campaña Las propiedades se utilizan para integrar Brightcove Players con plataformas de automatización de marketing y formularios de clientes potenciales. Para más información, ver Creación de un jugador habilitado para campaña.

Editor JSON

El Editor JSON se puede utilizar para editar la configuración del reproductor. Se debe tener cuidado al editar la configuración del reproductor.

Nota: Esta es una opción avanzada y solo debe ser utilizada por editores que se sientan cómodos editando la configuración del reproductor.

Para obtener información sobre cómo editar la configuración del reproductor, consulte la Guía de configuración del reproductor.

Análisis de terceros

La Análisis de terceros Las propiedades se utilizan para integrar Brightcove Players con Google Analytics y Adobe Analytics. Para más información, ver Configuración del seguimiento de eventos de Google Analytics y Configuración del seguimiento de eventos de Adobe Analytics.
