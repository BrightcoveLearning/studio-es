Uso del módulo de reproductores

Para acceder al módulo de reproductores, inicie sesión en Studio y haga clic en reproductores en el encabezado de navegación. Video Cloud Studio se muestra a continuación. La interfaz de usuario de Brightcove Player Studio tendrá pequeñas diferencias.

El cuerpo de la página mostrará:

  • Nombre del reproductor
  • ID de reproductor
  • Fecha y hora actualizadas
  • Estado del reproductor
    • - Indica que se han realizado cambios en el reproductor pero los cambios no se han publicado
    • - Indica que el reproductor está activado y todos los cambios realizados en el reproductor se han publicado
    • - Indica que el reproductor está inactivo

Con el módulo de reproductores, puede:

  • Crea un nuevo reproductor
  • Editar las propiedades del reproductor
  • Duplicar un reproductor
  • Eliminar reproductores
  • Buscar reproductores
  • Vista previa de un reproductor
  • Genere el código de inserción y la URL para un reproductor

Creando un nuevo reproductor

Para crear un nuevo reproductor usando el módulo de reproductores:

  1. Haga clic en el Nuevo reproductor botón.
  2. Ingrese un nombre y una breve descripción para el reproductor y luego haga clic en Ahorrar.
  3. El nuevo reproductor aparecerá en la lista de reproductores.
Nota: Se puede crear un máximo de 200 reproductores. Brightcove desalienta la creación de un gran número de reproductores pero, si es necesario, ponte en contacto con tu gestor de cuentas y solicita un aumento en el número de reproductores que puedes crear.

Cuando se crea un nuevo reproductor usando el módulo de reproductores, el reproductor se construye usando la última versión publicada por Brightcove. La versión del reproductor aparece en la página de inicio del módulo de reproductores. Tenga en cuenta que es posible que no todos los reproductores tengan la misma versión. A medida que Brightcove continúe desarrollando y mejorando el reproductor Brightcove, se lanzarán nuevas versiones del reproductor. De forma predeterminada, las actualizaciones se aplican a todos los reproductores automáticamente. Para evitar la actualización automática de un reproductor, debe configurar el Modo de actualizacion ajuste a Manual (también conocido como bloquear a un reproductor). Para más información, ver Configuración del modo y la versión de actualización del reproductor.

Editar las propiedades del reproductor

Existen numerosas propiedades del reproductor que se pueden configurar para personalizar la apariencia del reproductor:

  • Información del reproductor
  • Estilismo
  • Reproducción
  • Control S
  • Social
  • Complementos
  • Pantallas finales
  • Publicidad
  • Restricción de contenido
  • Campaña
  • Editor JSON
  • Análisis de terceros

Para editar las propiedades del reproductor, haga clic en un elemento del menú de navegación de la izquierda. Para obtener información sobre cómo configurar las propiedades del reproductor, consulte Configuración de las propiedades del reproductor.

Nota: Cualquier cambio realizado en un reproductor puede tardar hasta 5 minutos en aparecer en los reproductores publicados.

Duplicar un reproductor

Duplicar un reproductor le permite crear un nuevo reproductor basado en un reproductor existente. Para duplicar un reproductor en el módulo reproductores:

  1. Marque la casilla junto al reproductor que desea duplicar.
  2. Haga clic en el Duplicar botón.
  3. Entrar a Nombre y Breve descripción para el reproductor.
  4. Hacer clic Ahorrar.

Eliminando reproductores

Eliminar un reproductor lo elimina de su cuenta y no se puede deshacer. Para eliminar un reproductor en el módulo de reproductores:

  1. Marque la casilla junto a los reproductores que desea eliminar.
  2. Haga clic en el Borrar botón.
  3. Confirme la eliminación haciendo clic en Borrar.

Buscando reproductores

El campo de búsqueda se puede utilizar para buscar reproductores en su cuenta. Al buscar, se buscan los siguientes campos:

  • Nombre del reproductor
  • ID de reproductor
  • Breve descripción

Vista previa de un reproductor

Para obtener una vista previa de un reproductor, seleccione un reproductor de la lista de reproductores y luego haga clic en Vista previa del reproductor. El reproductor se abrirá en el cuadro de diálogo de vista previa.

Generando el código de inserción y la URL para un reproductor

La Insertar código y URL El botón puede generar el código de inserción del reproductor y la URL para el reproductor de vista previa o el reproductor publicado. Normalmente, el código de inserción y la URL del reproductor de vista previa se utilizan para obtener una vista previa de los cambios que se realizan en un reproductor. Esto le brinda la posibilidad de ver los cambios en acción antes de publicar el reproductor en un sitio de producción. Una vez que se publica un reproductor, los cambios se reflejarán en todas las instancias de ese reproductor en particular.

Para obtener información sobre las diferencias entre el reproductor de vista previa y el reproductor publicado y sobre cómo generar el código de inserción para un reproductor, consulte Generación de inserción de reproductor Código.

Atajos de teclado

Pulsando el ? en el módulo de reproductores abrirá una lista de las teclas de acceso directo que están disponibles.
