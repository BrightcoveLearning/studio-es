los lista de reproducción-pantalla final El complemento permite que se muestre una lista de reproducción de videos en la pantalla final del reproductor. Luego, el espectador puede hacer clic en la imagen en miniatura de un video para ver otro video. En este reproductor, se ha configurado una pantalla final de lista de reproducción.

Para obtener información sobre otras opciones para las pantallas finales del reproductor, consulte Configuración de la pantalla final del reproductor.

Nota: Para obtener información sobre cómo configurar la pantalla final de una lista de reproducción mediante programación, consulte la documentación del desarrollador.

Configuración del complemento de lista de reproducción-pantalla final

Para implementar el complemento de lista de reproducción-pantalla final mediante el módulo de reproductores, siga estos pasos:

  1. Abra el módulo de Jugadores y localice al jugador.
  2. Haga clic en el enlace del reproductor para abrir las propiedades del reproductor.
  3. Hacer clic Complementos en la navegación de la izquierda.
  4. Hacer clic Agregar un complemento> Complemento Brightcove.
  5. Para el Complemento Brightcove , Seleccione lista de reproducción-pantalla final.
  6. los Opciones (JSON) contiene la
    • playlistId - ID de la lista de reproducción para mostrar en la pantalla final
    • título (opcional): título para mostrar encima de la lista de videos
    7. {
  "playlistId": "1234567001",
  "title": "Related Videos"
}
  7. Hacer clic Ahorrar y entonces Publicar el jugador. Tenga en cuenta que los cambios pueden tardar hasta 5 minutos en aparecer en los reproductores publicados.
