Los reproductores se pueden configurar para que el contenido de video se pueda compartir en sitios populares de redes sociales. Cuando el uso compartido en redes sociales está habilitado, aparecerá un ícono de compartir en la barra de título y descripción.

Nota: Si la configuración del reproductor está configurada para no mostrar el título y la descripción, el icono de compartir aparecerá en la parte inferior del reproductor junto al cabezal de reproducción.

Cuando se hace clic en el icono para compartir, aparecerán las opciones para compartir seleccionadas que permiten compartir el video.

Para configurar el Social propiedades, siga estos pasos:

  1. Hacer clic reproductores en el encabezado de navegación para abrir el módulo reproductores.
  2. Haga clic en el enlace del reproductor que desea configurar.
  3. Hacer clic Social en el menú de navegación de la izquierda.
  4. Haga clic en la casilla de verificación para Habilitar compartir y seleccione los sitios de redes sociales en los que se puede compartir el video.
  5. Cheque Mostrar código de inserción del reproductor para permitir que el espectador copie el código de inserción del reproductor cuando se muestren las opciones para compartir en redes sociales.
  6. Cheque Mostrar enlace directo del reproductor para permitir que el espectador copie un enlace a la página donde está incrustado el reproductor cuando se muestran las opciones para compartir en redes sociales.
