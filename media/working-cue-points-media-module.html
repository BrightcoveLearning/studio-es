Un punto de referencia es un marcador en un momento preciso en la duración de un video. Puede utilizar puntos de referencia para activar anuncios mid-roll (es decir, anuncios que aparecen durante la reproducción del contenido del video, como una pausa comercial) o para separar capítulos o escenas en un video de formato largo.

Hay dos tipos de puntos de referencia, pero ambos simplemente proporcionan un valor de cadena al tipo propiedad del objeto de punto de referencia. El valor apropiado es útil solo si se usa en JavaScript personalizado mientras se procesa la información del punto de referencia. 

  • Anuncio - Asigna el valor de cadena de ANUNCIO hacia tipo propiedad
  • Código - Asigna el valor de cadena de CÓDIGO hacia tipo propiedad

Los puntos de referencia se pueden agregar a los videos usando el módulo de Medios o usando la API de Catálogo. Para obtener más información sobre cómo agregar puntos de referencia utilizando la API de catálogo, consulte el documento para desarrolladores Implementar puntos de referencia mediante programación.

Nota: No se pueden añadir puntos de referencia a los vídeos de activos remotos en directo.

Agregar puntos de referencia

Para agregar puntos de referencia a un video, abra el módulo de Medios y siga estos pasos:

  1. Haga clic en el enlace del título del video para abrir la página de propiedades del video.
  2. Utilice el reproductor de vista previa para reproducir el video. Pausa el video en el punto en el que deseas agregar un punto de referencia.
  3. Hacer clic Agregar punto de referencia.
  4. Si es necesario, ajuste el Código de tiempo (hh: mm: ss) valores para el punto de referencia. Tenga en cuenta que el segundos El campo admite milisegundos después del punto decimal.
  5. Selecciona el Tipo del punto de referencia y luego agregue los metadatos apropiados.
    • anuncio - Se realizará una solicitud de anuncio. Cuando se utilizan puntos de referencia de anuncios, el reproductor debe configurado para publicidad.
    • código - Los puntos de referencia se manejarán mediante programación. Ver el Implementar puntos de referencia mediante programación documento del desarrollador para obtener más información.
  6. Entrar a Nombre para el punto de referencia. Se puede acceder al valor del nombre programáticamente.
  7. Ingresar Metadatos (para puntos de referencia de código) o Pares clave / valor (para puntos de referencia de anuncios).
    • Metadatos - Metadatos opcionales que se almacenan con el punto de referencia. Se puede acceder a los metadatos programáticamente. Un ejemplo de cuándo puede querer usar un punto de referencia de código y pasar metadatos es cuando quiere mostrar diferentes llamadas a la acción mientras se reproduce el video. Ver el Muestra de jugador de Brightcove: Puntos de referencia Display CTA para un ejemplo.
    • Pares clave / valor - Pares clave / valor que se pueden transferir a una solicitud de anuncio. Un par clave-valor es un conjunto de dos elementos de datos vinculados: una clave, que es un identificador único para algún elemento de datos, y el valor, que son los datos que se identifican o un puntero a la ubicación de esos datos. Los pares clave / valor se pueden utilizar para ayudar a proporcionar anuncios más específicos. Todo lo que ingrese aquí se copiará en la etiqueta de anuncio donde el {mediainfo.ad_keys} se inserta la macro. Para más información, ver Publicidad con el complemento IMA3.

Después de agregar los puntos de referencia a un video, en tiempo de ejecución, los puntos de referencia se convierten en elementos de señal HTML5 WebVTT. Cuando se lleva a cabo esta conversión, algunos de los datos del punto de referencia, como el tipo y el tiempo del punto de referencia, se convierten en la pista de texto. El valor de tiempo del punto de referencia se asignará a la señal de WebVTT startTime y endTime propiedades. Ver este documento para obtener más información sobre la interfaz VTTCue.

Editar puntos de referencia

Para editar puntos de referencia en un video, abra el módulo de Medios y siga estos pasos:

  1. Haga clic en el enlace del título del video para abrir la página de propiedades del video.
  2. Haga clic en el punto de referencia para editar en la línea de tiempo.
    Nota: Una vez en el modo de edición, Anterior y próximo Los botones se pueden utilizar para navegar a otros puntos de referencia.
  3. Edite las propiedades del punto de referencia y luego haga clic en Ahorrar.
Nota: No es posible tener varios puntos de referencia con el mismo código de tiempo.

Eliminar puntos de referencia

Para eliminar puntos de referencia, abra el módulo de Medios y siga estos pasos:

  1. Haga clic en el enlace del título del video para abrir la página de propiedades del video.
  2. Haga clic en el punto de referencia para eliminarlo en la línea de tiempo.
  3. Haga clic en el icono Eliminar () junto a las propiedades del punto de referencia.
  4. Confirma la eliminación.
