El módulo de medios brinda la capacidad de exportar metadatos de video para todos sus videos o un subconjunto de videos. Cuando utilice esta función, tenga en cuenta lo siguiente:

  • Brightcove recomienda exportar menos de 15K videos a la vez. Si necesita exportar más de 15K videos, use la función de búsqueda para devolver un subconjunto de videos y exportar varios "lotes" a la vez.
  • Las URL para representaciones e imágenes no son fijas. Brightcove reconfigura el almacenamiento de activos multimedia de vez en cuando y, cuando esto sucede, las URL de activos específicos cambiarán. Si confía en URL codificadas de forma rígida para estos activos, los enlaces pueden romperse en algún momento. La mejor manera de evitar que los enlaces a los medios se rompan es recuperarlos de Video Cloud en tiempo de ejecución utilizando el API de CMS.
  • Los caracteres de dos bytes, como el coreano y el japonés, no se mostrarán correctamente si se abren en Excel. Esto se debe al hecho de que el archivo.csv generado está codificado por UTF-8 y Excel no lo admite correctamente. Para solucionar esto, ANSI debe volver a codificar el archivo.csv y, a continuación, se mostrará correctamente en Excel.

Para exportar metadatos de video, siga estos pasos.

  1. Abra el módulo de medios.
  2. Si desea exportar datos para videos específicos, marque la casilla de verificación junto a los videos.
  3. Hacer clic Exportar en la parte superior de la página.
  4. Seleccione los videos para exportar:
    • Todos los videos en esta lista - Exporta metadatos de video para todos los videos en la cuenta
    • x videos seleccionados - Exporta metadatos de video solo para los videos seleccionados
  5. Cheque Incluya las URL actuales para las representaciones de video para incluir las URL de reproducción de video. Marcar esta opción con una gran cantidad de videos seleccionados aumentará el tiempo requerido para generar el archivo de exportación.
    Nota: Cuando se selecciona esta opción, se pueden exportar metadatos de vídeo para un máximo de 1000 vídeos.
  6. Hacer clic Exportar datos. Comenzará la exportación de datos.
  7. Cuando el archivo de exportación esté listo, haga clic en Descargar archivo para descargar el archivo de datos de video a su computadora local.

El archivo.csv se nombrará con el número de cuenta de Video Cloud. Se puede utilizar un programa de hoja de cálculo de terceros para ver los datos de video exportados.

Detalles de datos de video exportados

La siguiente tabla describe los campos de datos de video que se exportan. Para obtener más información sobre los campos de datos de vídeo, consulte la Referencia de campos de video de la API de CMS.

Campo Descripción
identificación

ID de video
account_id

ID de cuenta de Video Cloud
ad_keys Cadena que representa los pares clave / valor del anuncio asignados al video.
clip_source_video_id ID de video de origen utilizado para crear este videoclip
completo Indica si el procesamiento está completo
created_at Cuándo se creó el video en el sistema Video Cloud
creado por Información sobre el usuario que creó el video:
  • si es creado por API, el type estarán api_key
  • si se creó antes de que se habilitara la función de auditoría el 7 de diciembre de 2018, type estarán unknown
cue_points Puntos de referencia de video
custom_fields Campos personalizados de video
tipo de entrega Define cómo se entrega el video
descripción Breve descripción del video
digital_master_id ID del sistema para el maestro digital
duración Duración del video en milisegundos
ciencias económicas "AD_SUPPORTED" (predeterminado) o "GRATIS"
folder_id ID de la carpeta a la que pertenece el video
geo Propiedades de restricción geográfica para el video
has_digital_master Si el video tiene un archivo maestro archivado que se puede usar para retranscodificar
imagenes Información sobre el póster y las imágenes en miniatura
vínculo Enlace relacionado
descripción larga Descripción larga del video
nombre Nombre del video
nombre_archivo_original Nombre de archivo original cuando se cargó el video
proyección La proyección cartográfica para videos de 360 °
publicado_en Fecha / hora en que se publicó el video
reference_id ID de referencia de video
calendario Mapa de fecha y hora de inicio y finalización para la disponibilidad del video
intercambio Mapa de las propiedades para compartir del video
estado ACTIVO, INACTIVO, PENDIENTE, ELIMINADO
etiquetas Matriz de etiquetas asignadas al video
text_tracks Matriz de pistas de texto (archivos WebVTT) asignadas al video
updated_at Fecha / hora en que se actualizó el video
updated_by Información sobre el usuario que actualizó el video por última vez:
  • si es creado por API, el type estarán api_key
  • si se creó antes de que se habilitara la función de auditoría el 7 de diciembre de 2018, type estarán unknown
  • si el vídeo no se modificó después de la ingestión, type el internal
offline_enabled Si el video está habilitado para reproducción sin conexión
