Brightcove se reserva el derecho de actualizar los perfiles de ingesta predeterminados asociados con las cuentas de Video Cloud. También puede tener la necesidad de actualizar sus propios perfiles de ingesta personalizados. Cuando cambian los perfiles de ingesta, es posible que desee volver a codificar los videos existentes en su cuenta para que tengan representaciones que se correspondan con la nueva configuración del perfil de ingesta.

Nota: La capacidad de retranscodificar videos existentes solo está disponible para cuentas Professional y Enterprise. Si está interesado en esta función, póngase en contacto con su gestor de cuentas.

Los videos se pueden volver a codificar usando el módulo de Medios o usando la API de Ingesta Dinámica. Utilizando el API de ingesta dinámica puede ser una mejor solución cuando tiene una gran cantidad de videos que requieren retranscodificación. Para volver a codificar videos con la API de ingesta dinámica, consulte la Descripción general de la API de ingesta dinámica documento de desarrollador.

Nota: No puedes volver a transcodificar vídeos que se han compartido en tu cuenta.
Nota: Solo se pueden volver a codificar los videos que se cargaron con el módulo de carga o con la API de ingesta dinámica. Si el vídeo no se puede retranscodificar, la retranscodificación fallará.
Nota: los videos solo se pueden volver a codificar si el video maestro se archivó cuando se agregó el video a la cuenta. Es posible que los vídeos más antiguos no tengan un maestro, según el perfil de ingesta que se haya utilizado.

Pasos para retranscodificar un video

Siga estos pasos para reemplazar las versiones existentes de un video.

  1. Abra el módulo de medios.
  2. Haga clic en el video que desea volver a codificar.
  3. En el Archivos de video sección, haga clic en Retranscodificar.
  4. Seleccione un perfil de ingesta para usar.
  5. Cheque Mantener las imágenes en miniatura y de póster existentes para mantener las imágenes existentes. Si desmarca esta casilla, se generarán nuevas imágenes basadas en la configuración de imagen en el perfil de ingesta seleccionado.
    Tenga en cuenta que si marca la casilla, las imágenes existentes se conservarán; sin embargo, las URL de la imagen cambiarán.
  6. Hacer clic Retranscodificar.

Mientras se crean las nuevas versiones, el video seguirá siendo reproducible con las versiones existentes.
