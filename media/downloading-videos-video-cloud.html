Hay varias formas de descargar videos desde una cuenta de Video Cloud. Los videos se pueden descargar usando los módulos de Medios o Sociales, usando las API y se puede agregar un enlace de descarga a un reproductor que permite a los espectadores descargar contenido.

Descarga de videos usando el módulo de medios

Puede descargar una versión de un video usando el módulo de Medios. Si el ícono de la nube junto al nombre del video en el módulo de Medios es gris (), el video se ingirió utilizando un perfil de ingesta heredado y tanto las versiones MP4 como las HLS se pueden descargar en el módulo de Medios. Un icono de nube verde () indica que el video fue ingerido usando Dynamic Delivery y si se seleccionó un perfil de ingesta que crea representaciones MP4, las representaciones MP4 se pueden descargar.

Notas:
  • La sección Archivos de video en la página de propiedades del video también indicará qué tipo de perfil de ingestión se usó.
  • Si desea tener una reproducción de 1080p disponible para descargar, necesitará crear un perfil de ingesta personalizado y asegúrese de que incluya al menos una de las siguientes versiones:
    • progresivo 3500
    • progresivo 4000

Para descargar un video usando el módulo de Medios, siga estos pasos:

  1. Abre el Medios de comunicación módulo.
  2. Haga clic en el enlace del video que desea descargar.
  3. Localizar el Archivos de video sección.
  4. Verifique el tipo de transcodificación que se usó:
    • Entrega dinámica (icono de nube verde, ejemplo a continuación) - Haga clic con el botón derecho en un enlace en el Dimensiones columna para descargar la versión; tenga en cuenta que solo se mostrará un enlace de descarga en el que se puede hacer clic si se han creado representaciones MP4
    • Legado (icono de nube gris): haga clic con el botón derecho en el enlace Formato columna para descargar la versión
Nota: Las representaciones generadas por Dynamic Delivery tienen un tiempo de vida (TTL) de 6 horas. Los enlaces generados tendrán una validez de 6 horas.

Es importante comprender que las URL de las representaciones e imágenes no son fijas. Brightcove reconfigura el almacenamiento de activos multimedia de vez en cuando y, cuando esto sucede, las URL de activos específicos cambiarán. Si confía en URL codificadas de forma rígida para estos activos, los enlaces pueden romperse en algún momento. La mejor manera de evitar que los enlaces a los medios se rompan es recuperarlos de Video Cloud en tiempo de ejecución utilizando el API de CMS.

Descarga de videos usando el módulo Social

El módulo Social brinda la capacidad de copiar la URL del video o descargar un archivo fuente de video. Al descargar, se descargará la reproducción MP4 de la más alta calidad.

Para descargar un video usando el módulo Social, siga estos pasos:

  1. Abra Brightcove Social.
  2. Hacer clic Videos en la navegación de la izquierda. Aparecerá una lista de videos en su cuenta de Video Cloud.
  3. Haga clic en el video para descargar y luego haga clic en Descargar.
    Nota: Si no hay versiones MP4, el Descargar El enlace se desactivará.
    Nota: No puede descargar videos que tienen DRM habilitado (aparece un candado en la miniatura del video). Si selecciona un video habilitado para DRM, el Descargar El botón se desactivará.
  4. Copie la URL del video o haga clic en el enlace para abrir el video en su navegador. Haga clic derecho en el video para guardar el archivo de origen localmente.
  5. Hacer clic Hecho.

Acceder a las propiedades del video usando la API de CMS

los API de CMS se puede utilizar para recuperar metadatos, incluidas las URL de video, para cualquier video en una cuenta de Video Cloud. Se requieren habilidades de desarrollo para utilizar la API de CMS. Para obtener información sobre cómo acceder a las propiedades de video mediante la API de CMS, consulte la Usando la API de CMS: Recuperar datos de video documento de desarrollador.

Brightcove Learning Services también ha creado una muestra API de CMS aplicación que puede utilizar para obtener enlaces de descarga: Ejemplo de API de CMS: Descargar enlaces.

Crear un enlace de descarga para archivos de video

Si tiene habilidades de desarrollo, es posible colocar un enlace de descarga en la barra de control del reproductor para dar a los espectadores la opción de descargar contenido.

También puede mostrar un enlace de descarga como una superposición del reproductor o debajo del reproductor. Para obtener más información sobre cómo hacer esto, incluido el código de muestra, consulte Muestra de reproductor de Brightcove: Descargar el complemento de video.
