Joomla es un popular sistema de administración de contenido (CMS) de código abierto, que le permite crear sitios web y potentes aplicaciones en línea. Puedes aprender más sobre Joomla aquí. Este tema le muestra cómo puede publicar fácilmente Brightcove Players en sus páginas de Joomla.

Nota: Los pasos e imágenes de este tema se han realizado con Joomla 3.9.6.

Para publicar un Brightcove Player en una página de Joomla, se requieren los siguientes pasos:

  • Compruebe la configuración del editor de Joomla
  • Pegar en el reproductor incrustado código

Comprobación de la configuración del editor de Joomla

De forma predeterminada, Joomla está configurado para usar el editor TinyMCE. El editor TinyMCE está configurado para eliminar ciertas etiquetas HTML cuando se pegan en el editor. Para pegar el código de publicación del reproductor estándar (iframe) en el editor de Joomla, debe asegurarse de que el <iframe> El editor no eliminará la etiqueta.

  1. Inicie sesión como administrador en su backend de Joomla.
  2. Seleccione Extensiones> Complementos.
  3. Ingresar tinymce en el cuadro de búsqueda y haga clic en el botón de búsqueda.
  4. Confirme que se devuelve un artículo, Editor - TinyMCE.
  5. Haga clic en el Editor - TinyMCE enlace del complemento para editar su configuración.
  6. Localizar el Elementos prohibidos sección y asegúrese de iframe no aparece en la lista. Si iframe aparece, elimínelo de la lista.
    Nota: Joomla requiere que haya un valor en el Elementos prohibidos campo. Si al quitar la iframe entrada se deja este campo en blanco, introduzca «ninguno» en este campo.
  7. Hacer clic Ahorrar en la parte superior de la página para guardar los cambios.

El editor de TinyMCE también eliminará la <script> etiqueta cuando se utiliza el código de publicación del reproductor avanzado (en la página). Una forma de evitar esto es configurar su instancia de Joomla para que no use el editor TinyMCE. Para seleccionar otro editor, siga estos pasos:

  1. Ir Sistema > configuración global.
  2. Haga clic en el Sitio pestaña.
  3. Encuentra el Editor predeterminado opción.
  4. Establecer esto opción a Editor - Ninguno. También hemos encontrado seleccionando Editor - CodeMirror no desnudará el <script> etiqueta.
  5. Hacer clic Guardar cerrar.

Si su instalación de Joomla está configurada para usar otro editor, debe asegurarse de que el editor no edite el código de publicación del reproductor cuando pegue y guarde el código en Joomla. Esto puede requerir que cambies otros ajustes de Joomla o, posiblemente, uses una extensión (Jumi o Sourcerer) para permitirte pegar código JavaScript en tu editor.

Pega el código de inserción del reproductor

Una vez que su editor esté configurado para que no edite el código, puede generar el código de inserción del reproductor.

  1. Usando el módulo de Medios, genere el código de inserción para el video que desea publicar.
  2. Abra la página que desea editar en Joomla.
  3. Haga clic en el Alternar editor en la parte inferior del editor para entrar en modo HTML.
  4. Pegue el código de publicación de su reproductor y haga clic en Guardar cerrar.
  5. Verifique que el video aparezca y se reproduzca en su página.
