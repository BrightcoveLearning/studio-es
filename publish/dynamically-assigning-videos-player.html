Es posible que haya casos en los que necesite cambiar el video asociado con un reproductor. Por ejemplo, tal vez el espectador tenga la capacidad de controlar qué video quiere ver. Quizás el video en el reproductor se basa en la parte del sitio de donde proviene el espectador.

Si necesita mostrar diferentes videos en un reproductor, se recomienda usar un reproductor y luego cambiar dinámicamente el video asociado con el reproductor.

En este tema se describen las siguientes técnicas:

  • Pasando el ID de video como un parámetro de URL
  • Pasando el ID de referencia como parámetro de URL
  • Modificar la inserción código para proporcionar una identificación de video
  • Modificar la inserción código para proporcionar una ID de referencia
Nota: Aunque es posible cambiar un video que está vinculado estáticamente a un reproductor, esta no es una buena práctica. Si hace esto, la imagen del póster del video enlazado estáticamente puede aparecer brevemente en el reproductor antes de que se muestre el video agregado dinámicamente. Por este motivo, es recomendable crear un reproductor que no tenga un vídeo asociado inicialmente a él para utilizarlo en situaciones en las que el vídeo se cambiará dinámicamente.

El contenido también se puede asignar dinámicamente a un reproductor mediante la API del catálogo. Para obtener más información y código de muestra, consulte el documento para desarrolladores Cambie dinámicamente los videos de origen.

Pasando la identificación del video como un parámetro de URL

Puede cambiar la fuente de video para un reproductor pasando el ID de video como un parámetro de URL usando el formulario ?videoId=YOUR_VIDEO_ID.

http://players.brightcove.net/20318290001/f1d70e07-8480-4bbd-b4ef-747e9333a034_default/index.html?videoId=4292158104001

Pasando el ID de referencia como parámetro de URL

Puede cambiar la fuente de video para un reproductor pasando el ID de referencia como un parámetro de URL usando el formulario ?videoId=ref:YOUR_VIDEO_REF_ID.

http://players.brightcove.net/20318290001/f1d70e07-8480-4bbd-b4ef-747e9333a034_default/index.html?videoId=ref:1234

Modificar el código de inserción para proporcionar una identificación de video

Puede asignar un video de Video Cloud al código de inserción del reproductor directamente usando la identificación de video.

Código incrustado en la página

Agregar el atributo data-video-id="YOUR_VIDEO_ID" hacia <video> etiqueta:

<video-js   data-account="57838016001"
  data-player="6e44f16e-31e9-48b2-83b9-6edd71217540"
  data-embed="default"
  data-video-id="68143720001"
  class="video-js"
  controls></video-js>
<script src="https://players.brightcove.net/57838016001/6e44f16e-31e9-48b2-83b9-6edd71217540_default/index.min.js"></script>
Nota: Cuando el reproductor JavaScript se ejecuta, lee el ID de video del atributo data-video-id solo al inicio. Cambiar el valor con, por ejemplo, el método setAttribute () de JavaScript, no hace que el reproductor cargue el video. Para ver un ejemplo sobre cómo cambiar el video usando JavaScript, consulte la Catálogo de jugadores documento de desarrollador.

Código de inserción de iframe

Agregar ?videoId=YOUR_VIDEO_ID al final de la URL en el src atributo para el iframe:

<iframe src='//players.brightcove.net/57838016001/6e44f16e-31e9-48b2-83b9-6edd71217540_default/index.html?videoId=68143720001'
 allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

Modificar el código de inserción para proporcionar una ID de referencia

Puede asignar un video de Video Cloud al código de inserción del reproductor directamente utilizando el ID de referencia.

Nota: Al utilizar un identificador de referencia que tenga caracteres especiales, debe escapar de los caracteres especiales del código de publicación o pueden surgir problemas.

Código incrustado en la página

Agregar el atributo data-video-id="ref:YOUR_VIDEO_REF_ID" hacia <video> etiqueta:

<video-js data-account="57838016001"
  data-player="6e44f16e-31e9-48b2-83b9-6edd71217540"
  data-embed="default"
  data-video-id="ref:68143720001"
  class="video-js"
  controls></video-js>
<script src="https://players.brightcove.net/57838016001/6e44f16e-31e9-48b2-83b9-6edd71217540_default/index.min.js"></script>

Código de inserción de iframe

Agregar ?videoId=ref:YOUR_VIDEO_REF_ID al final de la URL en el src atributo para el iframe:

<iframe src='//players.brightcove.net/57838016001/6e44f16e-31e9-48b2-83b9-6edd71217540_default/index.html?videoId=ref:68143720001'
 allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
