En la mayoría de las plataformas, no es posible insertar videos directamente en el correo electrónico. Esta no es una limitación de Brightcove, casi todos los sistemas de correo electrónico modernos eliminarán el código de inserción del reproductor de video por razones de seguridad. Sin embargo, puede generar conciencia y tráfico para el contenido creando un correo electrónico con un texto teaser y una imagen de póster. El módulo de medios ofrece la posibilidad de publicar un video en una página de destino de correo electrónico.

Nota: Para obtener otra opción para enviar vídeos por correo electrónico que proporciona un mayor control sobre la apariencia de la imagen del póster de vídeo y utiliza una página de destino personalizada, consulte Impulsar vistas de vídeo a través del correo electrónico.

Para publicar un video en una página de destino de correo electrónico, siga estos pasos:

  1. Abra el módulo de medios.
  2. Selecciona un video.
  3. Hacer clic Publicar e insertar> Página de destino de correo electrónico.
  4. Seleccione una página de destino. La lista desplegable contendrá un Página de destino simple que es una página de inicio de muestra sin estilo proporcionada por el sistema. La lista también contendrá las experiencias de página de destino publicadas que se crearon utilizando el módulo Galería. El uso de una página de inicio de la galería proporciona más flexibilidad sobre la apariencia y el estilo de la página de inicio. Los formularios de clientes potenciales también se pueden utilizar para capturar información del espectador. Para obtener más información sobre cómo crear páginas de destino de la Galería, consulte Creación de una experiencia de página de destino.
    Nota: La imagen del póster tendrá un tamaño de 640 x 360.
  5. Seleccionar para Reproducción automática de video. Esta opción solo está disponible al seleccionar el Página de destino simple.
  6. Seleccione un reproductor de video. Esta opción solo está disponible al seleccionar el Página de destino simple. Para las páginas de inicio de la Galería, el jugador se asigna como parte del experimentar propiedades.
  7. (Opcional) Haga clic en el Vista previa de la página de destino enlace para obtener una vista previa de la página de destino.
  8. Resalte la imagen, cópiela en el portapapeles y luego péguela en un mensaje de correo electrónico.

Hacer clic Generar enlace corto para generar un enlace abreviado que también se puede utilizar y distribuir. Cuando se recibe el correo electrónico, al hacer clic en la imagen del póster o en el enlace corto, se abrirá el video en una pestaña del navegador.

Si se utiliza una página de inicio de la Galería con un formulario de cliente potencial, los datos del formulario de cliente potencial se pueden descargar mediante el módulo Campaña. Para obtener información, consulte Exportación de datos de clientes potenciales de la galería.
