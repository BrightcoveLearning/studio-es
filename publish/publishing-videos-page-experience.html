Las experiencias en la página se crean utilizando el módulo Galería y permiten que uno o más videos, o una lista de reproducción, se publiquen en una página. Los videos pueden aparecer en carruseles, cuadrículas y listas de reproducción que cambian dinámicamente antes, durante y después de la reproducción del video. También se pueden agregar elementos interactivos, tanto dentro como fuera del reproductor de video. Estas experiencias en la página se pueden integrar fácilmente en sitios web, páginas de destino, blogs y campañas de correo electrónico.

Nota: Debe tener acceso al módulo Galería para poder publicar experiencias dentro de la página.
Nota: Para obtener una descripción general del trabajo con experiencias en la página, consulte Descripción general: Creación, edición y publicación de experiencias en la página.

El módulo de Medios le permite seleccionar uno o más videos, o una lista de reproducción, y luego creará una experiencia en la página que se puede personalizar y publicar usando el módulo de Galería.

Siga estos pasos para crear una experiencia en la página utilizando el módulo de medios.

  1. Abre el Medios de comunicación módulo.
  2. Seleccione los videos o la lista de reproducción para usar en la experiencia. Para seleccionar videos, haga clic en la casilla de verificación junto a los videos.

    Para seleccionar una lista de reproducción, haga clic en Todas las listas de reproducción en la barra de navegación izquierda y haga clic en la casilla de verificación junto a la lista de reproducción.

  3. Hacer clic Publicar e insertar> Experiencia en la página de la galería.
  4. Seleccione una plantilla para usar en la experiencia. En este ejemplo, el Red se selecciona la plantilla.

    Para obtener información sobre las plantillas en la página y ver algunos ejemplos, consulte Descripción general de las plantillas de experiencia en la página de la galería.

    Nota: Una vez seleccionada una plantilla, no se puede cambiar.
    Nota: Es posible que algunas plantillas no estén disponibles cuando se seleccionan una lista de reproducción o varios vídeos.
  5. Hacer clic próximo.
  6. Introduzca un título de experiencia, una descripción y elija un tamaño de diseño. Estos valores se pueden modificar posteriormente. Para obtener información sobre estos ajustes y cómo se utilizan, consulte Configurar los detalles para una experiencia en la página.
  7. Hacer clic Crear. El editor de experiencia en la página se abrirá en una nueva pestaña del navegador.
