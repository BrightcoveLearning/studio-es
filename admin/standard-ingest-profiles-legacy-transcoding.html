Cuando cargue videos usando el módulo Cargar o cuando cree una solicitud de Ingesta Dinámica usando la API DI, debe proporcionar un perfil de ingesta que determine el número y tipo de representaciones creadas por el proceso de ingestión de Video Cloud. Este tema proporciona información detallada sobre cada uno de los perfiles de ingesta heredados. Para obtener información sobre los perfiles de Dynamic Delivery, consulte Perfiles de ingesta estándar para entrega dinámica.

Nota: Cuando se carga contenido de origen de menor calidad (menor velocidad de bits) y se selecciona uno de los perfiles estándar, Video Cloud solo producirá copias de calidad igual o inferior a la fuente.

También es posible crear sus propios perfiles de ingesta personalizados. Para obtener información, consulte Gestión de perfiles de ingesta.
