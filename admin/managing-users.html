La página de Gestión de usuarios gestiona los usuarios autorizados para utilizar Studio. Tenga en cuenta que para acceder a la página de administración de usuarios, debe ser un usuario con el rol de administrador.

Para acceder a la página de Gestión de usuarios:

  1. Inicie sesión en Studio.
  2. En el encabezado de navegación, haga clic en ADMINISTRACIÓN y entonces Gestión de usuarios.

La página de Gestión de usuarios mostrará:

  • Apellido nombre
  • Dirección de correo electrónico
  • Rol: estándar o administrador
  • Acceso al módulo: cada módulo al que el usuario tiene acceso mostrará una marca de verificación verde ()
  • Último inicio de sesión: la última fecha y hora en que el usuario inició sesión en Video Cloud Studio. Este es el último inicio de sesión para todas las cuentas de Video Cloud Studio, no solo la cuenta actual.

La página de Gestión de usuarios se puede utilizar para:

  • Crear nuevos usuarios
  • Editar usuarios existentes
  • Eliminar un usuario
  • Cambiar el usuario principal

¿Cuántos usuarios puedo tener?

La cantidad de usuarios que puede crear depende del tipo de cuenta que tenga. Si necesita agregar más usuarios, puede actualizar su plan a uno que ofrezca más usuarios. El límite se aplica solo al número total de usuarios existentes en cualquier momento; si crea 10 usuarios, elimina 5 de ellos y crea 3 usuarios más, se considera que tiene 8 usuarios. La página Usuarios muestra a cuántos usuarios tiene derecho y cuántos usuarios adicionales puede crear.

Compra de usuarios adicionales

Si necesita agregar más usuarios, puede actualizar su plan a uno que ofrezca más usuarios. Las cuentas de nivel Pro y Enterprise deben comunicarse con su administrador de cuentas para obtener información sobre cómo agregar usuarios adicionales.

Creando nuevos usuarios

Para crear un nuevo usuario:

  1. Introduzca el correo electrónico, el nombre y el apellido del usuario.
  2. Asignar un rol al usuario:
    • Administrador - El usuario tiene acceso al ADMINISTRACIÓN menú (es decir, puede crear cuentas de usuario, perfiles de ingesta personalizados, campos personalizados, configurar el uso compartido de medios)
    • Estándar - El usuario no tiene acceso al ADMINISTRACIÓN menú
  3. Seleccione los módulos a los que el usuario tiene acceso. La desactivación de los módulos solo limita a qué secciones de la interfaz de usuario tiene acceso el usuario.
  4. Hacer clic Agregar usuario.

El usuario recibirá un mensaje de correo electrónico indicándole que ha sido agregado a la cuenta.

Editar usuarios existentes

Para editar un usuario existente:

  1. Haga clic en el usuario para editar. Los detalles del usuario aparecerán en la parte superior de la página.
  2. Edite los detalles del usuario.
    Nota: El usuario con el que ha iniciado sesión no se puede editar.
  3. Hacer clic Ahorrar para guardar los cambios.

Eliminar un usuario

Para eliminar un usuario:

  1. Haga clic en el icono de eliminar () asociado con el usuario.
  2. Hacer clic OK para confirmar la eliminación.

Cambiar el usuario principal

Un usuario en cada cuenta se designa como usuario principal. El usuario principal siempre tendrá la Administrador función y solo Brightcove Support puede eliminarlo.

Si necesita cambiar el usuario principal en algún momento, seleccione el nuevo usuario principal y luego haga clic en Hacer que este usuario sea el administrador principal de esta cuenta.

Se le pedirá que confirme el cambio de usuario principal.

Si el nuevo usuario principal tiene un rol de Estándar , se les asignará el Administrador rol cuando se convierte en el usuario principal.

Trabajar con varias cuentas

Un error común al agregar una cuenta adicional es que obtiene 10 usuarios adicionales tal como lo hizo con su primera cuenta de Video Cloud. Las cuentas de Video Cloud adicionales no incluyen cuentas de usuario adicionales. Por ejemplo, si se agregaron 10 usuarios a la primera cuenta y los mismos 10 usuarios se agregan a la segunda cuenta, el recuento total de usuarios sigue siendo 10. Los 10 usuarios únicos son según el acuerdo de licencia y una única identificación de usuario única se puede usar en varias cuentas y aún así contar como un usuario.

Si tiene varias cuentas, los usuarios deben agregarse a cada cuenta individualmente. Suponga que tiene 2 cuentas, Cuenta A y Cuenta B. Si se agrega un usuario a la Cuenta A, no tendrá acceso automáticamente a la Cuenta B. Los usuarios deben agregarse a cada cuenta a la que necesitan acceder.

Para obtener más información sobre cómo trabajar con varias cuentas, consulte Trabajar con varias cuentas de Video Cloud.
