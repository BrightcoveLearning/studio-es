Las configuraciones de anuncios definen varios aspectos de la reproducción de anuncios, incluidas las llamadas de anuncios, el tiempo, las balizas y otras opciones de configuración de anuncios. La configuración de anuncios del lado del servidor define los anuncios que se pueden solicitar durante los videos de video a pedido (VOD) y durante los eventos en vivo. Los anuncios del lado del servidor aprovechan la tecnología de inserción de anuncios del lado del servidor (SSAI) de Brightcove y proporcionan estas características principales:

  • Dado que los anuncios se unen a la transmisión de video a pedido o en vivo en el lado del servidor, no se ven afectados por los bloqueadores de anuncios.
  • Para eventos en vivo, los anuncios se insertan en los puntos de referencia enviados desde su codificador o puede crear un punto de referencia instantáneo usando el Sala de control del módulo en vivo
  • Puede configurar pizarras de relleno para llenar cualquier tiempo publicitario no utilizado

Para obtener más información sobre la publicidad del lado del servidor, consulte Implementación de anuncios del lado del servidor con Brightcove Player.

Nota:
  • Antes de poder utilizar SSAI, su cuenta de Video Cloud debe configurarse para Dynamic Delivery y habilitarse para SSAI. Comuníquese con su gerente de cuenta para obtener más detalles.
  • VMAP para Brightcove Live SSAI no es compatible actualmente.
  • Aunque no existe una función de captación previa para los anuncios SSAI, delegamos la lógica de la posición del anuncio en el proveedor de anuncios. Dynamic Delivery hace todas las llamadas adicionales por adelantado. Si obtenemos un anuncio del proveedor de anuncios y se transcodifica en nuestro sistema, lo insertaremos en la transmisión.

La página Configuración de anuncios del lado del servidor se utiliza para administrar las configuraciones de anuncios en vivo y VOD. Tenga en cuenta que para acceder a la página Configuración de anuncios del lado del servidor, debe ser un usuario con la función de administrador.

Para acceder a la página de configuración de anuncios del lado del servidor:

  1. Inicie sesión en Studio.
  2. En el encabezado de navegación, haga clic en ADMINISTRACIÓN y entonces Configuración de anuncios del lado del servidor. Se mostrará una lista de todas las configuraciones de anuncios.

Para crear una nueva configuración de anuncios, haga clic en Crear configuración de anuncios y luego haga clic en el tipo específico de anuncio para crear:

  • Video a pedido (VOD)
  • Vivir

Creación de una configuración de anuncios de video a pedido

Nota: Las configuraciones de anuncios del lado del servidor también se pueden administrar mediante la API de inserción de anuncios del lado del servidor. Para obtener información, consulte el API de configuración de anuncios SSAI de Video Cloud tema.

Para agregar una configuración de anuncios de video a pedido (VOD), siga estos pasos.

  1. En la página Configuración de anuncios del lado del servidor, haga clic en Crear configuración de anuncios> VOD.
  2. Entrar a Nombre.
  3. Selecciona el Respuesta del servidor de anuncios:
    • Reglas de anuncios de DFP
    • DFP VMAP
    • XML inteligente
    • VASTA 3.0
  4. Introduzca la etiqueta de anuncio.
  5. Cheque Deshabilitar las balizas del lado del servidor para deshabilitar las balizas del lado del servidor. Cuando está deshabilitado, SSAI no activará ninguna baliza del lado del servidor e incluirá todas las balizas en la salida VMAP.
  6. Hacer clic Ahorrar.

Para editar la configuración de un anuncio, haga clic en el nombre del anuncio, realice los cambios y luego haga clic en Ahorrar. Para eliminar una configuración de anuncio, marque la casilla junto al anuncio y luego haga clic en Borrar. Para duplicar una configuración de anuncio, marque la casilla junto al anuncio y luego haga clic en Duplicar.

Una vez que se hayan creado las configuraciones de anuncios VOD, abra el módulo Reproductores y haga clic en un reproductor para ver las propiedades. Cuando el Habilitar el lado del servidor (SSAI) La propiedad publicitaria está habilitada, se puede seleccionar una configuración de anuncios. Para obtener información sobre cómo configurar la publicidad del reproductor, consulte Configurar la publicidad de los reproductores mediante el módulo de reproductores.

Crear una configuración de anuncios en vivo

Notas:
  • Las configuraciones de anuncios en vivo del lado del servidor también se pueden administrar utilizando la API en vivo. Para obtener información, consulte API de Brightcove Live: Inserción de anuncios en el servidor (SSAI).

Para agregar una configuración de anuncio en vivo, siga estos pasos.

  1. En la página Configuración de anuncios del lado del servidor, haga clic en Crear configuración de anuncios> En vivo.
  2. Entrar a Nombre.
  3. Selecciona el Respuesta del servidor de anuncios:
    • VASTO
    • DFP (reglas publicitarias)
    • XML inteligente
  4. Introduzca la etiqueta de anuncio.
  5. Hacer clic Ahorrar.

Para editar la configuración de un anuncio, haga clic en el nombre del anuncio, realice los cambios y luego haga clic en Ahorrar. Para eliminar una configuración de anuncio, marque la casilla junto al anuncio y luego haga clic en Borrar. Para duplicar una configuración de anuncio, marque la casilla junto al anuncio y luego haga clic en Duplicar.

Una vez que se han creado las configuraciones de anuncios en vivo, cuando se crea un evento con el Habilitar la inserción de anuncios del lado del servidor (SSAI) habilitada, se puede seleccionar una configuración de anuncios en la Sala de control. Para obtener información sobre la creación de eventos en vivo, consulte Creación y administración de eventos en vivo usando el módulo en vivo.
