Se pueden usar metadatos personalizados para agregar campos de metadatos adicionales a sus videos. Los campos personalizados se agregan como parte de la configuración de su cuenta. Una vez que se han agregado los campos personalizados, se pueden asignar valores a los campos mediante el módulo Medios.

Creando campos personalizados

Para crear campos personalizados, siga estos pasos.

Nota: Después de realizar cambios en los campos personalizados, es posible que deba borrar la caché del navegador y volver a cargar la página para ver los cambios.
  1. Inicie sesión en Video Cloud Studio.
  2. En el encabezado de navegación, haga clic en ADMINISTRACIÓN y entonces Campos de video. Tenga en cuenta que debe ser un administrador de cuenta para acceder al menú ADMIN.
  3. La página Campos de video mostrará sus campos personalizados y todos los campos de metadatos de video estándar. Revise los campos estándar antes de crear nuevos campos personalizados. Esto le ayudará a evitar la creación de campos personalizados cuando ya exista un campo estándar que puede hacer el trabajo.
  4. Hacer clic Agregar campo personalizado.
  5. Entrar a Nombre para mostrar. Esto aparecerá como la etiqueta del campo en los módulos de Carga y Medios. Un Nombre interno se creará automáticamente para el campo según el nombre para mostrar. El nombre interno se utiliza cuando se accede al campo mediante programación mediante cualquiera de las API de Video Cloud. El nombre interno se puede cambiar, pero el nombre interno no puede incluir espacios en blanco, caracteres de doble byte o puntuación.

    Tenga en cuenta que los nombres internos nunca tienen espacios y están todos en minúsculas. Por lo general, se forman poniendo el nombre para mostrar en minúsculas y reemplazando los espacios con caracteres de subrayado.

    Por ejemplo, si ingresa My Field como nombre de visualización, el nombre interno será my_field.

    Nota: Hay algunas palabras reservadas que no debe utilizarse como nombres internos de campos personalizados (el nombre de visualización no importa). Ver el Nombres reservados sección para obtener información.
  6. Seleccione un Tipo para el campo.
    • Texto - El usuario puede escribir cualquier cosa en el campo hasta 1024 caracteres de un solo byte (o 512 caracteres de doble byte).
    • Lista - El usuario seleccionará de una lista de valores. Las listas no pueden tener más de 150 valores posibles y cada valor de la lista no puede tener más de 100 caracteres.
  7. Entrar a Descripción. El campo de descripción aparecerá como una sugerencia en la interfaz de Studio.
  8. Si se requerirá el campo personalizado, marque Hacer que este campo sea obligatorio.
  9. Si el tipo de campo es Lista, introduzca una lista de valores separada por comas y, a continuación, haga clic en Agregar a la lista.

    Notas:
    1. Los valores no deben contener comillas (ejemplo: Animales, Aves, Peces).
    2. Ningún valor puede contener una coma; todas las comas se tratarán como separadores de diferentes valores.
    Nota: Si los valores de los campos personalizados tienen menos de tres caracteres, no poder buscar videos basados en ese valor, ya sea en Studio o usando la API de CMS. Por ejemplo, si desea crear un campo personalizado tipo booleano isfeatureitem, haga la lista de valores true y false, no yes y no; si utilizas estos últimos valores, no se pueden encontrar vídeos mediante la búsqueda isfeatureitem:no.
  10. Hacer clic Guardar campo para guardar los cambios. Hacer clic Guardar y agregar otro para guardar el campo y comenzar a crear otro campo.

Los campos personalizados que cree están disponibles de inmediato y será visible en el módulo de Medios.

Se pueden crear hasta 10 campos personalizados. Si necesita más de 10 campos, haga clic en el ¿Necesitas más campos? en la página Campos de vídeo para ponerse en contacto con el servicio de atención al cliente de Brightcove para solicitar más. Los editores están limitados a 50 campos personalizados.

Editar valores de lista

Puede agregar valores adicionales a un campo personalizado de tipo Lista. Haga clic en el enlace del nombre del campo personalizado, agregue los valores y luego haga clic en Agregar a la lista. Cuando termine, haga clic en Guardar campo.

Eliminar valores de lista

Puede agregar eliminar valores existentes de un campo personalizado de tipo Lista. Haga clic en el enlace del nombre del campo personalizado, haga clic en el icono de eliminar () asociado con el valor y luego haga clic en Guardar campo.

Cuando se elimina un valor de lista, los videos a los que se les asignó previamente ese valor mantendrán el valor. Sin embargo, si se edita un video con el valor eliminado, el valor se borrará y se deberá seleccionar un nuevo valor.

Eliminar campos personalizados de una cuenta

Para eliminar un campo personalizado de su cuenta, debe póngase en contacto con el soporte de Brightcove. Tenga en cuenta que si Brightcove Support elimina los campos personalizados, la fecha de modificación cambiará para los videos en los que haya un valor.

Nombres reservados

Hay algunas palabras reservadas que no debe utilizarse como nombres internos de campos personalizados (el nombre de visualización no importa). Un nombre común es categoría. Si usa un nombre reservado como nombre interno para un campo personalizado, las búsquedas en ese campo no arrojarán resultados. Si ya tiene un campo personalizado llamado que usa un nombre reservado, debe eliminarlo y reemplácelo con un nombre diferente o comuníquese con el Soporte de Brightcove para obtener orientación.

Los siguientes no deben usarse como nombres de campos personalizados internos.

categoría completo created_at
fecha de creación custom_fields importación de datos
discriminado nombre para mostrar displayname_analyzed
displayname_tokenized fecha final estado de artículo
lastmodifieddate mediaid nombre
jugable_ahora playstotal semana de juego
publicado_en fecha de publicación editor
id de referencia referenceid_analyzed programado
referenceid_analyzed schedule.ends_at schedule.starts_at
programado buscar texto searchtext_analyzed
fecha de inicio etiqueta etiquetas
texto estado
