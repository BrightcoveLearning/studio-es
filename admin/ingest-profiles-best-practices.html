¿Cuántos perfiles debería tener?

Puede definir hasta 200 perfiles personalizados. Sin embargo, por lo general, debe crear solo los que necesite para evitar confusiones al seleccionar perfiles para trabajos de ingestión.

¿Cómo administro mis perfiles?

Puedes usar el Página de gestión de perfiles de ingesta en Studio para gestionar sus perfiles de ingesta. Hay una variedad de opciones disponibles dentro de esta interfaz, que incluyen:

  • Configuración de su perfil predeterminado
  • Creando nuevos perfiles
  • Clonación de perfiles existentes
  • Editar perfiles personalizados
  • Eliminar perfiles innecesarios

Si necesita acceso programático a sus perfiles de ingesta, puede crear, leer, actualizar y eliminar perfiles personalizados usando el API de REST de perfiles de ingesta. Tenga en cuenta que esto requiere que configure Credenciales de OAuth.

¿Cuántas versiones puede tener un perfil?

A Entrega dinámica El perfil puede tener hasta 37 representaciones. Tenga en cuenta que para los perfiles personalizados de Dynamic Delivery, solo puede seleccionar el representaciones predefinidas que están disponibles actualmente.

Si está creando un perfil personalizado de codificación sensible al contexto, min_renditions tiene un valor mínimo de 1, y max_renditions tiene un valor máximo de 15.

Tenga en cuenta que las representaciones innecesarias se sumarán a sus costos de almacenamiento administrado, por lo que no debe crear representaciones que probablemente no se reproducirán dado el tipo de videos que publica y las audiencias y dispositivos a los que se dirige.

Los perfiles estándar de su cuenta están diseñados para proporcionar buenas experiencias de visualización en los anchos de banda típicos de los espectadores y los dispositivos más utilizados.

Nombrar perfiles personalizados

Los nombres de perfil personalizados no deben incluir espacios ni caracteres especiales. Utilice letras y números, y reemplace los espacios con guiones (-) o guiones bajos (_).

Para evitar errores del usuario, intente nombrar los perfiles de una manera que comunique claramente a las personas que suben videos para qué se debe usar el perfil.

¿Qué perfil debo establecer como predeterminado?

Cualquier perfil estándar o personalizado se puede establecer como predeterminado en Studio o mediante la API de perfil de ingesta. El valor predeterminado solo se usará si el usuario no especifica un perfil en particular (ya sea en el módulo de carga o usando la API de ingesta dinámica).

Recomendamos que establezca el perfil predeterminado en uno que no sea una mala elección si el usuario se olvida de especificar un perfil en particular.
