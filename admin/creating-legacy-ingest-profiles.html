La página de administración de Ingest Profiles se puede utilizar para crear nuevos perfiles de ingesta heredados. Los perfiles de ingesta se definen mediante JSON. Al crear un nuevo perfil heredado, es útil revisar el Perfiles de ingesta estándar para la transcodificación heredada Documente y busque un perfil similar al que necesita. Luego, en Studio, puede duplicar fácilmente el perfil y luego editarlo con sus cambios.

Nota: Para obtener información sobre la creación de perfiles de ingesta que utilizan DRM, consulte la Vídeo MPEG-DASH documento de desarrollador.

Nota: Cada perfil puede crear un máximo de 25 representaciones.  Los pósters y las imágenes en miniatura cuentan para el máximo de 25. 

Para crear un nuevo perfil de ingesta heredado:

  1. Haga clic en el ADMINISTRACIÓN enlace en el encabezado de Studio.
  2. Haga clic en el Perfiles de ingesta Enlace.
  3. Busque un perfil heredado para duplicarlo. Haga clic en el nombre del perfil para abrir el editor de perfiles.
  4. Hacer clic Perfil duplicado. El editor de perfiles se abrirá con el nuevo perfil.
  5. Desplácese hasta la parte superior del perfil. Edite el name elemento para darle al perfil un nombre significativo. Este nombre se mostrará dentro de Video Cloud Studio.
  6. Edite el description elemento para darle al perfil una descripción significativa.
  7. Edite la información de la reproducción según sea necesario. El tamaño predeterminado para el póster y las imágenes en miniatura se encuentra al final del perfil.
  8. Hacer clic Ahorrar.

Agregar una marca de agua a un perfil

Se puede agregar una marca de agua a cada interpretación agregando el watermarks elemento a cada rendition:

       marcas de agua ": [{ "y": -20, "x": -10, "width": "20%", "url": "http://www.mysite.com/companylogo.png" }]

La watermarks elemento tiene los siguientes parámetros.

Parámetro Tipo Descripción
url cuerda URL de la imagen de la marca de agua
width cuerda ancho en píxeles o porcentaje del ancho del marco; por ejemplo, 20 o 10%
height cuerda altura en píxeles o porcentaje del ancho del marco; por ejemplo, 20 o 10%
x cuerda distancia desde el borde izquierdo hasta el centro de la imagen como píxeles o porcentaje del ancho del marco; por ejemplo, 20 o 10% (use valores negativos como -10 para colocar la marca de agua en relación con el borde derecho)
y cuerda distancia desde el borde superior al centro si la imagen se expresa como píxeles o porcentaje de la altura del marco; por ejemplo, 20 o 10% (utilice valores negativos como -10 para colocar la marca de agua en relación con el borde inferior)
