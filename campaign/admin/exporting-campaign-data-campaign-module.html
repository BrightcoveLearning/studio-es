Los eventos de visualización se registran a medida que se miran los videos con reproductores habilitados para Campaña. Los datos de los clientes potenciales se capturan cuando los usuarios ingresan datos en los formularios de clientes potenciales de Campaign. El módulo Campaña ofrece dos tipos de formularios de clientes potenciales:

  • Formularios de clientes potenciales de campaña - Estos formularios se crean seleccionando de un conjunto de campos predefinidos en el módulo Campaña. Estos formularios guardan los datos de los clientes potenciales en Video Cloud. Para obtener información sobre cómo crear formularios de clientes potenciales de campaña, consulte Creación de un formulario de cliente potencial de campaña.
  • Formularios de clientes potenciales personalizados - Estos formularios se crean utilizando su plataforma de automatización de marketing. El código de inserción del formulario se utiliza luego en el módulo Campaña para crear un formulario de cliente potencial personalizado. Estos formularios envían datos de clientes potenciales a la plataforma de automatización de marketing. Para obtener información sobre cómo crear formularios de clientes potenciales, consulte el documento de su plataforma de marketing:
    • Eloqua
    • Marketo
    • Fuerza de ventas
    • HubSpot

Al exportar datos de clientes potenciales desde el módulo Campaña, solo se exportarán los datos capturados mediante los formularios de clientes potenciales de Campaña. Los datos de clientes potenciales de formularios de clientes potenciales personalizados se almacenarán en su software de automatización de marketing.

Para exportar datos, siga estos pasos:

  1. Abra el módulo Campaña.
  2. Hacer clic Exportar campaña Datos en la navegación de la izquierda.
    Nota: Debe ser administrador de cuentas para acceder al menú Exportar datos.
  3. Seleccione un Conección de datos para exportar datos. La Conección de datos El menú desplegable mostrará una lista de todas las integraciones que se han configurado.
  4. Seleccione los datos para exportar.
    • Ver eventos - Grabado por jugadores habilitados para la campaña
    • Dirige - Capturado mediante formularios de clientes potenciales
  5. Utilice el selector de fecha para seleccionar un Fecha de inicio y Fecha final.
  6. Seleccione un Formato de exportación.
    • Valores separados por comas (CSV)
    • Excel (XLS)
  7. Hacer clic Exportar para descargar los datos del cliente potencial.
  8. El formulario de cliente potencial y los datos del evento de visualización se descargarán en archivos separados. Dirige Los datos contendrán los datos que se ingresaron en el formulario de cliente potencial, así como los siguientes campos:
    • externalId - Esta es la ID externa que aparece en la página Toda la actividad reciente. En el caso de Oracle Eloqua, es la guía de Eloqua. Con Marketo, es la guía de Marketo
    • playerId - ID de jugador del jugador que se estaba utilizando cuando se mostró el formulario
    • videoId - ID de video del video que se estaba reproduciendo cuando se mostró el formulario
    • PageURL: URL de página que muestra el formulario de clientes potenciales (solo experiencias de la galería)
    • createdAt: fecha y hora en que se capturaron los datos
    • experiencia - Experiencia de la galería que recopiló los datos (solo experiencias de la galería)

    Ver eventos Los datos contendrán los siguientes campos:

    • externalId - Esta es la ID externa que aparece en la página Toda la actividad reciente. En el caso de Oracle Eloqua, es la guía de Eloqua. Con Marketo, es la guía de Marketo
    • videoId - ID de video del video
    • videoName - Nombre del video
    • watchched - Porcentaje de vídeo visto (al porcentaje más cercano)
    • timeWatched - Tiempo total visto
    • playerId - ID de jugador del jugador utilizado para ver el video
    • PageURL - URL de la página que contiene el reproductor
    • createdAt: fecha y hora en que se capturaron los datos
    • actualizado en - Fecha y hora en que se guardó por última vez el evento, que se traduce aproximadamente a cuando el espectador dejó de mirar; si el video tiene solo un minuto de duración, createdAt y updatedAt estará muy cerca
    • isSynced - Especifica si este evento de vista se ha exportado automáticamente a la herramienta de automatización de marketing del editor.
    • TrackinID: ID de seguimiento pasado como parámetro de URL. Para obtener información sobre cómo trabajar con un ID de seguimiento, consulte Creación de integraciones personalizadas con el módulo de campaña.
    • Campos UTM - utmSource, utmMedium, utmCampaign, utmTerm y utmContent. Para obtener información sobre cómo trabajar con campos UTM, consulte Trabajar con campos UTM en el módulo de campaña.
    • metric1-3, event1-3 - Se utiliza cuando se utilizan métricas y eventos personalizados
