Los datos del formulario de clientes potenciales de la Galería se recopilan cuando se ha configurado una experiencia de Galería para usar un formulario de clientes potenciales. Para obtener información sobre cómo configurar una experiencia de portal para recopilar datos de clientes potenciales, consulte Uso de un formulario de cliente potencial con una experiencia de portal.

Al exportar clientes potenciales de la Galería desde el módulo Campaña, solo se exportarán los datos capturados mediante los formularios de clientes potenciales de Campaña. Los datos de clientes potenciales de formularios de clientes potenciales personalizados se almacenarán en su software de automatización de marketing.

Nota: Solo los usuarios con privilegios de administrador podrán descargar los datos del cliente potencial.

Para exportar datos de clientes potenciales de la Galería, siga estos pasos.

  1. Abre el Campaña módulo.
  2. Hacer clic Exportar clientes potenciales de la galería en la navegación de la izquierda.
  3. Seleccione una experiencia de Galería desde la que exportar los clientes potenciales.
  4. Entrar a Fecha de inicio y Fecha final para los clientes potenciales.
  5. Seleccione un formato de exportación, Valores Separados por Comas o Sobresalir.
  6. Haga clic en el Exportar para descargar los datos del cliente potencial.

Los siguientes campos se incluirán en el archivo exportado:

  • Todos los campos posibles del formulario de cliente potencial
    • dirección de correo electrónico
    • primer nombre
    • apellido
    • Teléfono de negocios
    • país
    • nombre de empresa
    • industria
  • externalId - ID utilizado para identificar de forma única al espectador
  • playerId - ID de reproductor que mostraba el formulario de cliente potencial
  • videoId - ID de video del video que se estaba viendo cuando se recopilaron los datos
  • PageURL: el visor de URL de página estaba activado cuando se recopilaron los datos
  • CreatedAt: se recopilaron datos de fecha y hora
  • experiencia - Nombre de la experiencia que se muestra cuando se recopilaron los datos
