La opción administrar datos del visor le permite buscar un usuario y luego eliminar ese usuario o exportar los datos de ese usuario.

Notas:
  • Por lo general, cuando se integra con una plataforma de automatización de marketing (integración MAP), la configuración del período de retención de datos se usa para controlar cuánto tiempo se retienen los datos.
  • Debe ser un administrador de cuenta para acceder al menú Administrar datos del espectador

Para administrar los datos de los espectadores, siga estos pasos.

  1. Abra el módulo Campaña.
  2. Hacer clic Administrar datos de espectadores en la navegación de la izquierda.
  3. Ingrese el usuario a buscar. Normalmente, este será uno de los siguientes valores:
    • Correo electrónico - Cuando se usa el seguimiento de campañas y se usa un formulario de clientes potenciales de campaña o con integraciones de SSO
    • ID de usuario - Al usar SSO o integraciones personalizadas
    • GUID personalizado - Para usuarios anónimos
    • GUID externo - Cuando se utiliza una integración MAP
  4. Si se encuentra una coincidencia, seleccione una acción:
    • Eliminar datos del espectador - Eliminar datos de visualización y clientes potenciales para el usuario
    • Exportar datos del visor - Exportar el cliente potencial y ver los datos del evento del usuario. Los datos del cliente potencial contendrán los datos que se ingresaron en un formulario de cliente potencial de la campaña. Los datos del evento de visualización contienen todos los eventos de visualización del espectador.
