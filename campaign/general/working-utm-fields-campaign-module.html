Los códigos UTM (también conocidos como parámetros UTM) son simples fragmentos de texto que puede agregar a una URL para ayudar a rastrear el éxito de su contenido web. Hay 5 parámetros estándar que puede agregar a cualquier URL:

  • Campaña
  • Fuente
  • Medio
  • Contexto
  • Término

Los códigos UTM se agregan comúnmente a las URL personalizadas. A continuación se muestra un ejemplo de códigos UTM (en negrita) agregados a una URL.

https://www.brightcove.com/en/blog/2018/08/define-your-video-strategy-3-simple-steps-1?utm_campaign=blogpost&utm_medium=social&utm_source=LinkedIn

Los códigos UTM le brindan la capacidad de rastrear cómo funciona una publicación de blog, un anuncio de Internet o un envío masivo de correo electrónico sin tener que crear páginas de destino personalizadas para cada campaña. Al crear un código UTM distinto para publicaciones de blogs, anuncios y correos electrónicos, puede obtener datos sobre los que generan más tráfico, conversiones, etc.

Configurar conexiones de datos para trabajar con campos UTM

Campaign analiza los campos UTM a partir de los argumentos de la URL. A continuación, se puede acceder a los datos UTM desde su software de automatización de marketing o utilizando la API Audience. La configuración de campaña que se requiere se basa en la plataforma de automatización de marketing que se utiliza.

  • HubSpot
  • Fuerza de ventas
  • Eloqua
  • Marketo

HubSpot

Con las conexiones de datos de HubSpot, no se requiere ninguna configuración adicional. Los campos UTM aparecerán automáticamente.

Fuerza de ventas

Con conexiones de datos de Salesforce, use un Tipo de mapeo de Avanzado y luego los campos UTM se pueden asignar a los objetos de datos apropiados en Salesforce.

Eloqua

Con las conexiones de datos de Eloqua, no se requiere ninguna configuración adicional. Los campos UTM aparecerán automáticamente.

Marketo

Debido al hecho de que no hay suficientes campos que podamos agregar a la actividad personalizada que creamos en Marketo, los campos UTM no son compatibles con las conexiones de Marketo.

Exportación de datos mediante el módulo Campaña

Los campos UTM se incluyen al exportar datos de campaña y seleccionar Ver eventos. Para obtener información sobre la exportación de datos, consulte Exportación de datos de campaña desde el módulo de campaña.

Accediendo a los campos UTM usando la API Audience

Se puede acceder a los campos UTM mediante la API de audiencia. Para obtener información, consulte Documentación de la API de audiencia.
