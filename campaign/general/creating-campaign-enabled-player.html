Los datos de participación de video solo se registrarán cuando el contenido de video se vea dentro de un Brightcove Player habilitado para Campaign. Hay dos formas de crear un reproductor habilitado para campaña:

  • Crea un nuevo jugador usando el módulo Campaña
  • Habilitar Camapign en un jugador existente

Creando un nuevo jugador usando el módulo Campaña

Antes de crear un reproductor habilitado para Campaña usando el módulo Campaña, el módulo Campaña debe configurarse con al menos una conexión de datos:

  • Eloqua
  • Marketo (Munchkin)
  • Marketo (API REST)
  • Fuerza de ventas
  • HubSpot (API REST)
  • HubSpot (API del lado del cliente)
  • SSO
  • Integración personalizada

Tenga en cuenta que Seguimiento de campañas se proporciona de forma predeterminada. Para crear un reproductor habilitado para Campaña usando el módulo Campaña, siga estos pasos.

  1. Abra el módulo Campaña.
  2. Hacer clic Jugadores de campaña en la navegación de la izquierda.
  3. Hacer clic Crear jugador. Se creará un nuevo jugador habilitado para la campaña y aparecerá en la lista de jugadores.
  4. Utilizando la Conectar a menú desplegable, seleccione la conexión de datos que utilizará el reproductor.
  5. (Opcional) Utilizando la Formulario de cliente potencial desplegable, seleccione el forma de plomo que utilizará el jugador.

Habilitando Campaña en un jugador existente

Para habilitar la campaña a un jugador existente, siga estos pasos.

  1. Abre el Jugadores módulo.
  2. Haga clic en el nombre de un jugador para abrir las propiedades del jugador.
  3. Hacer clic Campaña en el menú de navegación de la izquierda.
  4. Comprobar el Habilitar campaña opción.
  5. Utilizar el Conectar a menú desplegable para seleccionar una conexión de campaña.
  6. (Opcional) Utilice el Formulario de cliente potencial desplegable para seleccionar un forma de plomo usar.
  7. Hacer clic Ahorrar. Aparecerá un mensaje indicando que tiene cambios no publicados.
  8. Hacer clic Publicar e insertar y entonces Publicar cambios para publicar el cambio en el reproductor.
  9. Regrese al módulo Campaña. Hacer clic Jugadores de campaña en la navegación de la izquierda.
  10. Confirma que el jugador aparece en la lista de jugadores de la campaña.
