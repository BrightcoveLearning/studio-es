Cuando se utilizan integraciones de seguimiento de campañas, los perfiles de los espectadores se crearán en función de los datos que se utilizaron para identificar al espectador. Los perfiles de espectador le permiten ver los datos de visualización de un espectador determinado.

Se realizará un seguimiento de todas las visualizaciones de video en los reproductores habilitados para la campaña. Para ver la actividad del video, haga clic en Perfil de espectador (no MAP) en la navegación de la izquierda. La página Perfiles de espectadores mostrará una lista de toda la actividad de visualización de las integraciones de seguimiento de campañas para el período de tiempo seleccionado. El control de calendario se puede utilizar para seleccionar un nuevo rango de fechas. El valor del Visor será el valor que se utilizó para identificar al usuario. Normalmente, esta será la dirección de correo electrónico capturada en un formulario de cliente potencial. Los datos se pueden ordenar haciendo clic en los encabezados de las columnas.

Nota: Habrá un retraso de unas horas antes de que aparezcan los datos de visualización.

Al hacer clic en el enlace de un espectador, se mostrará toda la actividad de visualización de ese espectador. Los datos se pueden ordenar haciendo clic en los encabezados de las columnas.

Los datos se pueden exportar haciendo clic en el CSV o XLS botones en la parte superior de la página.

Otras opciones de visualización de datos

Los datos de la campaña se pueden ver / exportar de otras formas:

  • Actividad reciente - Se utiliza para ver datos de eventos de visualización de video durante las últimas 24 horas. Para obtener información, consulte Visualización de eventos de reproductores en el módulo de campaña.
  • Actividad de video - Se usa para ver la actividad del espectador por video. Para obtener información, consulte Visualización de la actividad de video.
  • Exportar datos - Se utiliza para exportar vista de video y datos de clientes potenciales. Para obtener información, consulte Exportación de datos de campaña desde el módulo de campaña.
  • API de audiencia - La API Audience se puede utilizar para recuperar de manera programática la vista de video y los datos de clientes potenciales. Para obtener información, consulte Descripción general: API de audiencia.
