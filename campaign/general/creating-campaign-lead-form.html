Los formularios de clientes potenciales se pueden utilizar para capturar información del espectador durante la reproducción de video. Hay dos tipos de formularios de clientes potenciales:

  • Formularios de clientes potenciales de campaña - Le permite seleccionar de un conjunto de campos y los datos del formulario se guardan localmente en Video Cloud. Estos formularios se utilizan con jugadores que están configurados con seguimiento de campañas.
  • Formularios de clientes potenciales personalizados - Le permite ingresar un formulario HTML personalizado generado a partir de una plataforma de automatización de marketing. Estos formularios se utilizan con jugadores que están configurados con una conexión a una plataforma de automatización de marketing.

Para obtener más información sobre la creación de formularios de clientes potenciales personalizados con una plataforma de automatización de marketing, consulte el documento de su plataforma:

  • Oracle Eloqua
  • Marketo
  • HubSpot
  • Fuerza de ventas

Crear un formulario de cliente potencial

Para crear un formulario de cliente potencial, siga estos pasos.

Configurar el contenido del formulario

  1. Abra el módulo Campaña.

  2. Hacer clic Formularios de clientes potenciales en la navegación de la izquierda. Si se han creado formularios de clientes potenciales, se mostrarán.
  3. Hacer clic Crear formulario de cliente potencial.
  4. Dale a la forma de plomo un descriptivo Nombre.
  5. Seleccione un Estado del formulario de cliente potencial.
    • Activo
    • Inactivo
  6. Seleccione un Lenguaje de formulario.
  7. Selecciona el Tiempo de formulario. El tiempo del formulario determina cuándo se mostrará el formulario principal.
  8. Cheque Requerir envío para ver el video para solicitar que se envíe el formulario de cliente potencial para ver un video.
  9. Comprobación Muestre siempre el formulario de cliente potencial incluso si el cliente potencial ya es conocido siempre mostrará el formulario de cliente potencial incluso si ya se conoce el cliente potencial. Si no se marca, el formulario de cliente potencial no se muestra si se conoce el cliente potencial.
  10. Seleccione el tipo de formulario.
    • Formulario de cliente potencial de campaña - El formulario se crea con un diseño y un conjunto de campos predeterminados, los datos se guardan localmente en Video Cloud
    • Formulario de cliente potencial personalizado - Proporciona la capacidad de ingresar HTML personalizado (generalmente desde una plataforma de automatización de marketing), los datos se guardan en la plataforma de automatización de marketing
  11. Entrar a Mensaje superior. El mensaje superior es el que se mostrará en la parte superior del formulario de cliente potencial.
  12. Seleccione los campos que se mostrarán en el formulario de cliente potencial. Se pueden seleccionar un máximo de 4 campos.
  13. Introducir el Texto de la política de privacidad y URL de la política de privacidad. Estos campos son obligatorios para guardar el formulario de cliente potencial.
  14. Introduzca el texto del botón que se utilizará para la Cancelar y Botón de enviar.
  15. (Opcional) Hacer clic Formulario de vista previa para abrir una vista previa del formulario. El cuadro de diálogo de vista previa se puede utilizar para obtener una vista previa del formulario en dispositivos móviles, tabletas o de escritorio. Hacer clic Cerca cuando termine.
  16. Hacer clic Ahorrar para guardar el formulario de cliente potencial.

El formulario de cliente potencial se puede editar haciendo clic en el Editar botón.

Asignar una forma de cliente potencial a un jugador

Para asignar un formulario de cliente potencial a un jugador, siga estos pasos:

  1. Hacer clic Jugadores de campaña en la navegación de la izquierda.
  2. Localice su reproductor y utilice el Formulario de cliente potencial lista desplegable para seleccionar el formulario de cliente potencial.

La configuración ahora está completa y está listo para capturar información de clientes potenciales. Cuando se publica un video con un reproductor habilitado para Campaña, el formulario de cliente potencial aparecerá de acuerdo con la configuración de tiempo del formulario.

Exportación de datos de formularios de clientes potenciales

Hay dos opciones para exportar datos de clientes potenciales a los que se accede desde el menú de navegación de la izquierda:

  • Exportar datos de campaña - Se utiliza para exportar datos de clientes potenciales recopilados de Brightcove Players habilitados para la campaña. Para obtener información, consulte Exportación de datos de campaña desde el módulo de campaña.
  • Exportar clientes potenciales de la galería - Se utiliza para exportar datos de clientes potenciales recopilados de las experiencias de la Galería. Para obtener información, consulte Exportación de datos de clientes potenciales de la galería.
