Al usar el seguimiento de campañas o las integraciones de SSO, el módulo de campaña le permite seleccionar un video y luego ver la actividad de ese video. Esto le permite ver quién vio un video en particular.

Se realizará un seguimiento de todas las visualizaciones de video en los reproductores habilitados para la campaña. Para ver la actividad del video, haga clic en Video (no MAP) en la navegación de la izquierda. La página Ver actividad mostrará una lista de todos los videos vistos de SSO y las integraciones de seguimiento de campañas para el período de tiempo seleccionado. La Seleccionar conexión El control se puede utilizar para seleccionar una conexión específica. El control de calendario se puede utilizar para seleccionar un nuevo rango de fechas. Los datos se pueden ordenar haciendo clic en los encabezados de las columnas.

Notas:
  • Habrá un retraso de algunas horas antes de que aparezcan los datos de visualización.
  • Seguimiento de campañas y análisis a través de complementos independientes y sistemas de backend. Es muy común no ver los mismos resultados. La campaña también se adhiere a cosas como no rastrear la configuración en el navegador. Para obtener los análisis más completos, consulte los informes del módulo de análisis.

Al hacer clic en el enlace de un video, se mostrará toda la actividad de visualización de ese video. Los datos se pueden ordenar haciendo clic en los encabezados de las columnas.

Los datos se pueden exportar haciendo clic en el CSV o XLS botones en la parte superior de la página.

Otras opciones de visualización de datos

El módulo Campaña proporciona varias formas de ver / exportar datos:

  • Actividad reciente - Se utiliza para ver datos de eventos de visualización de video durante las últimas 24 horas. Para obtener información, consulte Visualización de eventos de reproductores en el módulo de campaña.
  • Perfiles de espectadores - Se utiliza para ver la actividad de video por espectador. Para obtener información, consulte Visualización de la información del perfil del espectador.
  • Exportar datos - Se utiliza para exportar vista de video y datos de clientes potenciales. Para obtener información, consulte Exportación de datos de campaña desde el módulo de campaña.
  • API de audiencia - La API Audience se puede utilizar para recuperar de manera programática la vista de video y los datos de clientes potenciales. Para obtener información, consulte Descripción general: API de audiencia.
