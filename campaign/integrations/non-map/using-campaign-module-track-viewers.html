El módulo Campaña ofrece la posibilidad de realizar un seguimiento de la actividad de visualización. En este escenario, los datos de visualización no se envían a una plataforma de automatización de marketing. Un caso de uso común para esto serían los videos de capacitación internos. El seguimiento de campañas se puede utilizar para realizar un seguimiento de la actividad de visualización de los empleados internos. Para utilizar el seguimiento de campañas, se requieren los siguientes pasos:

  1. Configure los ajustes de seguimiento de la campaña
  2. Configurar un reproductor y un formulario de cliente potencial para el seguimiento de campañas
  3. Ver datos de video

Configurar los ajustes de seguimiento de campañas

Para configurar el seguimiento de campañas, siga estos pasos:

  1. Abra el módulo Campaña y luego haga clic en Conexiones de datos en la navegación de la izquierda.
  2. Localizar el Seguimiento de campañas sección y haga clic en Show.
  3. De forma predeterminada, el seguimiento de campañas está habilitado para todas las cuentas. Edite la configuración según sea necesario:
    • Nombre de la conexión - Identifica la conexión
    • Estado de conexión - Determina si el módulo Campaña guarda los datos de participación de video de Video Cloud.
      • Activado - Se recopilan datos
      • Discapacitado - No se recopilan datos (la conexión está deshabilitada)
    • Seguimiento de datos para espectadores desconocidos - Determina si los datos de seguimiento se registran para espectadores desconocidos. Para identificar a los espectadores cuando se utiliza el seguimiento de campañas, un Formulario de cliente potencial de campaña puede ser usado.
      Nota: Con el seguimiento de campañas, se realizará un seguimiento de hasta 1.000 espectadores por periodo de 24 horas.
    • ¿Cuánto tiempo debe conservar Brightcove los datos? - La política de retención de datos determina cuánto tiempo se conservarán los datos de visualización y los datos del formulario de cliente potencial (si utiliza un formulario de cliente potencial de la campaña) en su cuenta de Video Cloud. Debe ajustar la configuración de retención de acuerdo con su política de privacidad; no es necesario retener ningún dato de usuario en Video Cloud.

Configuración de un reproductor y formulario de cliente potencial para el seguimiento de campañas

La actividad de visualización solo se puede rastrear cuando los videos se ven dentro de un reproductor de video habilitado para Campaña. Debe utilizar un reproductor habilitado para campaña cuando utilice el seguimiento de campaña. Los formularios de clientes potenciales se pueden mostrar durante la reproducción de video para capturar información del espectador. Los formularios de clientes potenciales no son necesarios cuando se utiliza el seguimiento de campañas.

Para configurar el seguimiento de campañas con un reproductor y un formulario de cliente potencial habilitados para la campaña, siga estos pasos:

  1. Hacer clic reproductores de campaña en la navegación de la izquierda.
  2. Determina qué reproductor habilitado para campaña usar. Si no existen reproductores habilitados para la campaña, haga clic en Crear reproductor para crear uno. Para obtener más información sobre la creación de reproductores habilitados para la campaña, consulte Creación de un reproductor habilitado para campaña.
  3. Si bien no se requiere un formulario de clientes potenciales, los formularios de clientes potenciales se pueden utilizar para recopilar información sobre el espectador y convertir a los espectadores anónimos en espectadores conocidos. Hacer clic Formularios de clientes potenciales y crear un formulario de cliente potencial. Para obtener más información sobre cómo crear formularios de clientes potenciales, consulte Creación de un formulario de cliente potencial de campaña.
  4. Hacer clic reproductores de campaña.
  5. Busque el reproductor habilitado para la campaña que se va a utilizar. Selecciona el Conectar a valor para que el reproductor Seguimiento de campañas.
  6. Si se utiliza un formulario de cliente potencial, seleccione el formulario de cliente potencial de la Formulario de cliente potencial la lista desplegable.

Ver datos de video

Hay varias opciones disponibles para ver datos de video cuando se usa Campaign Tracking:

  • Actividades del espectador (eventos) - Proporcionada como parte del módulo Campaña, esta opción se puede utilizar para ver datos de video dentro del módulo Campaña. Para obtener información, consulte Visualización de eventos de reproductores en el módulo de campaña.
  • Perfiles de espectadores - Proporcionada como parte del módulo Campaña, esta opción se puede usar para ver datos de video basados en un valor de perfil de usuario dentro del módulo Campaña. Para obtener información, consulte Visualización de la información del perfil del espectador.
  • Exportar datos - Proporcionada como parte del módulo Campaña, esta opción se puede utilizar para exportar la vista de video y los datos de clientes potenciales. Para obtener información, consulte Exportación de datos de campaña desde el módulo de campaña.
  • API de audiencia - La API Audience se puede utilizar para recuperar de manera programática la vista de video y los datos de clientes potenciales. Para obtener información, consulte Descripción general: API de audiencia.
